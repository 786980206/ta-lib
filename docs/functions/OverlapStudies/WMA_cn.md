# WMA - 加权移动平均线

## 描述

加权移动平均线（WMA）是一种对近期数据点给予更多权重，对较旧数据点给予较少权重的移动平均线。与对所有数据等权重的简单移动平均线（SMA）不同，WMA使用线性加权方案，其中最近的值具有最高权重。

## 类别
重叠研究

## 计算

在WMA中，每个价格都乘以一个加权因子。最近的价格具有最高权重（n），下一个具有（n-1），依此类推：

### 公式
```
WMA = (P1 × n + P2 × (n-1) + P3 × (n-2) + ... + Pn × 1) / (n + (n-1) + (n-2) + ... + 1)
```

或者更简洁地：
```
WMA = Σ(价格[i] × 权重[i]) / Σ(权重[i])
```

其中：
- n = 周期
- P1 = 最近价格
- Pn = 周期内最旧价格
- 权重总和 = n × (n+1) / 2

### 示例
对于5周期WMA，价格为[10, 11, 12, 13, 14]：
```
WMA = (14×5 + 13×4 + 12×3 + 11×2 + 10×1) / (5+4+3+2+1)
    = (70 + 52 + 36 + 22 + 10) / 15
    = 190 / 15
    = 12.67
```

## 参数

- **optInTimePeriod**（默认：30）：加权移动平均线的周期数
  - 有效范围：2到100000
  - 常用值：10, 20, 50

## 输入
- 价格数据：`double[]`（通常是收盘价）

## 输出
- WMA值：`double[]`

## 解释

### 与SMA和EMA比较
- **比SMA更敏感**：对价格变化反应更快
- **比EMA不太敏感**：比指数移动平均线慢
- **可预测加权**：线性权重衰减直观

### 用途
与其他移动平均线相同：
- **趋势方向**：斜率表示趋势
- **支撑/阻力**：作为动态水平
- **交叉**：生成交易信号

## 使用示例

```c
// C/C++示例
double closePrices[100];
double wmaOutput[100];
int outBegIdx, outNBElement;

// 计算20周期WMA
TA_RetCode retCode = TA_WMA(
    0,                    // 开始索引
    99,                   // 结束索引
    closePrices,          // 输入价格数据
    20,                   // 时间周期
    &outBegIdx,           // 输出：开始索引
    &outNBElement,        // 输出：元素数量
    wmaOutput             // 输出：WMA值
);
```

## 实现细节

TA-Lib WMA实现使用高效算法：

1. **加权总和**：计算价格的加权总和
2. **除数**：使用三角数n×(n+1)/2
3. **滑动计算**：为每个新K线高效更新
4. **回望**：在第一个输出之前需要period-1个K线

## 优势

1. **更敏感**：比SMA更敏感，更好地捕捉近期变化
2. **比EMA更平滑**：比指数移动平均线震荡更少
3. **线性逻辑**：加权方案直观
4. **平衡**：SMA和EMA之间的良好中间地带

## 局限性

1. **仍滞后**：比SMA少但比EMA多
2. **不太流行**：不如SMA或EMA广泛使用
3. **有限记忆**：与EMA不同，完全丢弃旧数据
4. **计算**：比SMA稍微复杂

## 移动平均线比较

| 类型 | 加权 | 响应性 | 滞后 | 平滑性 |
|------|------|--------|------|--------|
| SMA | 等权重 | 低 | 高 | 高 |
| WMA | 线性 | 中等 | 中等 | 中等 |
| EMA | 指数 | 高 | 低 | 低 |
| DEMA | 双指数 | 非常高 | 非常低 | 非常低 |

## 相关函数

- **SMA**：简单移动平均线 - 等权重
- **EMA**：指数移动平均线 - 指数加权
- **DEMA**：双指数移动平均线 - 减少滞后
- **TEMA**：三指数移动平均线 - 最小滞后
- **TRIMA**：三角移动平均线 - 双重平滑

## 附加说明

WMA在SMA的平滑性和EMA的响应性之间提供了良好的折衷。它特别适用于想要对近期数据给予更多权重但更喜欢线性、可预测的加权方案而不是EMA指数方法的交易者。

加权方案具有直观意义：如果你看10周期WMA，今天的价格对10天前价格的影响是10倍。这种逐渐的、线性的权重减少比EMA的指数衰减更容易理解。
