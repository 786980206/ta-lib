# SMA - 简单移动平均

## 描述

简单移动平均（SMA）是最基本的移动平均类型。它计算指定周期内价格的算术平均值。SMA平滑价格数据以通过过滤短期波动来识别趋势。

## 分类
重叠研究

## 计算方法

SMA通过将指定周期内的价格求和并除以周期数来计算：

### 公式
```
SMA = (P1 + P2 + P3 + ... + Pn) / n
```

其中：
- P1, P2, ..., Pn是每个周期的价格（通常是收盘价）
- n是周期数

### 示例
对于收盘价为[10, 11, 12, 13, 14]的5日SMA：
```
SMA = (10 + 11 + 12 + 13 + 14) / 5 = 60 / 5 = 12
```

## 参数

- **optInTimePeriod**（默认：30）：移动平均的周期数
  - 有效范围：2到100000
  - 常用值：10、20、50、100、200

## 输入
- 价格数据：`double[]`（通常是收盘价）

## 输出
- SMA值：`double[]`

## 解释

### 趋势识别
- **上升趋势**：价格高于SMA，SMA向上倾斜
- **下降趋势**：价格低于SMA，SMA向下倾斜
- **交叉**：价格交叉SMA可以信号趋势变化

### 交易信号
1. **价格交叉**：
   - **买入信号**：价格交叉SMA上方
   - **卖出信号**：价格交叉SMA下方

2. **移动平均交叉**（使用多个SMA）：
   - **金叉**：较短SMA（例如50日）交叉较长SMA（例如200日）上方 - 看涨
   - **死叉**：较短SMA交叉较长SMA下方 - 看跌

3. **支撑和阻力**：
   - SMA可以在上升趋势中充当动态支撑
   - SMA可以在下降趋势中充当动态阻力

### 常见周期
- **10日SMA**：非常短期，对价格变化敏感
- **20日SMA**：短期趋势
- **50日SMA**：中期趋势
- **100日SMA**：中期趋势
- **200日SMA**：长期趋势（机构投资者最关注）

## 使用示例

```c
// C/C++ 示例
double closePrices[100];
double smaOutput[100];
int outBegIdx, outNBElement;

// 计算20周期SMA
TA_RetCode retCode = TA_SMA(

    99,                   // 结束索引
    closePrices,          // 输入价格数据
    20,                   // 时间周期
    &outBegIdx,           // 输出：开始索引
    &outNBElement,        // 输出：元素数量
    smaOutput             // 输出：SMA值

```

## 实现细节

TA-Lib实现使用高效的滑动窗口算法：

1. **初始和**：计算前`period`个值的和
2. **滑动窗口**：对于每个后续值：
   - 将新值添加到和中
   - 从和中减去最旧的值
   - 除以周期

这种方法计算效率高，复杂度为O(n)。

### 特征
- **回望周期**：在产生第一个输出之前需要`timePeriod - 1`个柱
- **等权重**：周期内所有价格具有相等权重
- **滞后**：SMA比指数移动平均滞后价格行为更多

## 优势

1. **计算简单**：易于理解和计算
2. **平滑**：有效过滤短期噪音
3. **广泛使用**：技术分析中的标准，易于与他人比较
4. **客观**：计算中无主观解释

## 限制和考虑

1. **滞后指标**：本质上，移动平均滞后价格行为
2. **等权重**：将旧数据与最近数据同等对待，这可能不理想
3. **鞭打**：在波动、横盘市场中可能产生假信号
4. **非预测性**：仅描述过去的价格行为，不预测未来移动
5. **周期敏感性**：结果因所选周期而异

## 与其他移动平均的比较

| 移动平均 | 权重 | 响应性 | 平滑性 |
|---------|------|--------|--------|
| SMA | 等权重 | 低 | 高 |
| EMA | 指数 | 高 | 中等 |
| WMA | 线性 | 中等 | 中等 |
| DEMA | 双指数 | 非常高 | 低 |
| TEMA | 三指数 | 非常高 | 低 |

## 相关函数

- **EMA**：指数移动平均 - 给最近价格更多权重
- **WMA**：加权移动平均 - 线性权重方案
- **DEMA**：双指数移动平均 - 对价格变化响应更快
- **TEMA**：三指数移动平均 - 响应更快
- **TRIMA**：三角移动平均 - 双重平滑
- **KAMA**：Kaufman自适应移动平均 - 适应市场波动性

## 附加说明

简单移动平均是最古老和最广泛使用的技术指标之一。它构成许多其他技术分析工具和策略的基础。虽然由于滞后性和等权重而有限制，但它仍然非常有效：

- 识别整体趋势方向
- 提供支撑和阻力水平
- 平滑价格数据以看到更大的图景
- 作为更复杂指标的组成部分

200日SMA在金融市场中特别重要，被机构投资者和交易者密切关注，作为长期趋势健康的关键指标。当主要指数在其200日SMA上方交易时，通常被认为是牛市；在其下方被认为是熊市。
