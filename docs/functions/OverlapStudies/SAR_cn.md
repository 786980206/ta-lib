# SAR - 抛物线SAR

## 描述

抛物线SAR（止损和反转）是J. Welles Wilder开发的趋势跟踪指标，提供入场和出场点。SAR代表"止损和反转"，意味着当指标给出信号时，它建议关闭当前头寸并在相反方向开新头寸。该指标显示为价格上方或下方的点，随着趋势继续逐渐接近价格。

## 分类
重叠研究/趋势跟踪

## 作者
J. Welles Wilder

## 计算方法

抛物线SAR计算复杂且迭代：

### 基本概念
- **SAR点在价格上方**：下降趋势（空头头寸）
- **SAR点在价格下方**：上升趋势（多头头寸）

### 公式
```
SAR(明天) = SAR(今天) + AF × (EP - SAR(今天))
```

其中：
- **AF** = 加速因子（从0.02开始，每次EP创新高/低时增加0.02，最大值通常0.20）
- **EP** = 极值点（上升趋势中的最高价，下降趋势中的最低价）
- **SAR** = 止损和反转点

### 计算规则
1. **开始**：SAR在重要高点（下降趋势）或低点（上升趋势）初始化
2. **AF增量**：创新EP时增加
3. **SAR限制**：不能高于前两个低点（上升趋势）或低于前两个高点（下降趋势）
4. **反转**：当价格触及SAR时，趋势反转

## 参数

- **optInAcceleration**（默认：0.02）：起始加速因子
  - 有效范围：0到3.0e+37
  - 标准：0.02

- **optInMaximum**（默认：0.20）：最大加速因子
  - 有效范围：0到3.0e+37
  - 标准：0.20

## 输入
- **最高价**：`double[]`
- **最低价**：`double[]`

## 输出
- SAR值：`double[]`（止损的价格水平）

## 解释

### SAR位置
- **SAR在价格下方**：上升趋势，持有多头头寸
- **SAR在价格上方**：下降趋势，持有空头头寸
- **价格触及SAR**：反转头寸

### 交易信号
1. **买入信号**：SAR从价格上方翻转到下方
2. **卖出信号**：SAR从价格下方翻转到上方
3. **跟踪止损**：使用SAR水平作为止损

### 加速因子
- **低AF（0.02-0.10）**：更宽的止损，较少反转，趋势跟踪
- **高AF（0.15-0.30）**：更紧的止损，更多反转，更快出场

## 使用示例

```c
// C/C++ 示例
double high[100], low[100];
double sarOutput[100];
int outBegIdx, outNBElement;

// 计算抛物线SAR
TA_RetCode retCode = TA_SAR(
    0,                    // 开始索引
    99,                   // 结束索引
    high,                 // 最高价
    low,                  // 最低价

    &outBegIdx,           // 输出：开始索引
    &outNBElement,        // 输出：元素数量
    sarOutput             // 输出：SAR值
);
```

## 实现细节

TA-Lib SAR实现：

1. **初始化**：确定初始趋势方向
2. **AF管理**：跟踪和增加加速因子
3. **EP跟踪**：创新高/低时更新极值点
4. **SAR计算**：应用带约束的公式
5. **反转检测**：识别价格何时触及SAR

### 复杂规则
- SAR不能反转到前柱的范围内
- AF在每次反转时重置为0.02
- EP在反转时重置
- 处理多个边缘情况

## 交易策略

### 1. 纯SAR策略
- **多头**：当SAR在价格下方时
- **空头**：当SAR在价格上方时
- **反转**：当SAR翻转时
- **止损**：在SAR水平
- **最适合**：强趋势市场

### 2. SAR + 趋势过滤器
- **过滤器**：使用ADX或MA进行趋势确认
- **入场**：仅在趋势确认时使用SAR信号
- **出场**：SAR反转或趋势过滤器出场
- **最适合**：避免横盘市场

### 3. SAR + 动量
- **设置**：SAR确定方向
- **确认**：RSI或MACD确定动量
- **入场**：当两者对齐时
- **出场**：SAR反转
- **最适合**：高概率趋势入场

### 4. SAR跟踪止损
- **入场**：使用其他指标入场
- **止损**：在SAR水平放置止损
- **跟踪**：随着SAR移动移动止损
- **出场**：当SAR被触及或其他信号时
- **最适合**：在趋势中保护利润

## 优势

1. **清晰信号**：明确的入场/出场点
2. **始终在市场**：持续的多头或空头头寸
3. **跟踪止损**：自动跟踪止损水平
4. **视觉**：在图表上易于看到
5. **客观**：无需解释
6. **趋势跟踪**：非常适合捕获趋势

## 限制

1. **鞭打**：在横盘市场中频繁反转
2. **滞后**：跟随价格，不预测
3. **固定逻辑**：不适应波动性变化
4. **不适合横盘市场**：在横盘市场中表现差
5. **始终在头寸**：需要主动管理
6. **参数敏感性**：结果因AF设置而异

## 参数调整

### 加速因子（AF）
- **保守（0.01，0.10）**：
  - 更宽的止损
  - 较少反转
  - 更适合长期趋势
  - 响应性较低

- **标准（0.02，0.20）**：
  - Wilder的原始设置
  - 平衡方法
  - 最常用

- **激进（0.03，0.30）**：
  - 更紧的止损
  - 更多反转
  - 更适合短期交易
  - 更具响应性

### 市场特定调整
- **波动市场**：降低AF以避免鞭打
- **趋势市场**：标准或更高AF
- **横盘市场**：避免SAR或使用非常低的AF

## 市场应用

### SAR的最佳市场
- **趋势股票**：效果极佳
- **商品**：最初设计用于此
- **外汇主要货币对**：通常良好
- **指数**：在强趋势中效果良好

### SAR的不良市场
- **横盘市场**：产生许多假信号
- **低流动性**：缺口可能触发过早反转
- **非常波动的市场**：太多反转

## 相关函数

- **SAREXT**：抛物线SAR扩展 - 更多参数
- **ADX**：平均方向指数 - 趋势强度过滤器
- **ATR**：平均真实范围 - 可用于止损放置

## 附加说明

J. Welles Wilder在1978年引入了抛物线SAR。它是他最受欢迎的指标之一，与RSI和ATR并列。"抛物线"名称来自SAR点向价格加速时的弯曲形状。

### 关键洞察

1. **止损和反转哲学**：
   - 始终在市场（多头或空头）
   - 不适合所有交易者/策略
   - 原始商品方法
   - 现代交易者经常仅用于止损

2. **加速是关键**：
   - SAR随着趋势继续而加速
   - 代表跟踪止损收紧
   - 在强移动中强制获利
   - 防止突然反转

3. **视觉力量**：
   - 最具视觉性的指标之一
   - 点清楚地显示趋势和止损
   - 初学者易于理解
   - 无需复杂解释

4. **组合策略**：
   - SAR单独产生太多信号
   - 最好与趋势过滤器结合（ADX、MA）
   - 用于头寸管理，不仅仅是入场
   - 作为跟踪止损机制有效

5. **SAREXT vs. SAR**：
   - SAR：标准，简单参数
   - SAREXT：扩展版本，更多控制
   - SAREXT允许AF的不同起始值
   - 大多数交易者使用标准SAR

### 实用技巧

**对于趋势跟踪**：
- 使用标准参数（0.02，0.20）
- 与ADX > 25过滤器结合
- 当ADX < 25时忽略SAR信号
- 在SAR反转时获利

**对于止损放置**：
- 在其他信号上入场
- 在SAR水平放置止损
- 随着SAR移动移动止损
- 永远不要逆着头寸移动止损

**对于波动市场**：
- 将最大AF降低到0.15或0.10
- 减少反转次数
- 更宽的止损适应波动性
- 较少鞭打

**对于安静市场**：
- 可以将最大AF增加到0.25-0.30
- 更紧的止损合适
- 对较小移动更具响应性
- 反转时快速出场

抛物线SAR在用作趋势跟踪系统并带有额外过滤器以避免横盘市场时最有效，或作为更广泛交易策略中的跟踪止损机制。
