# TRIMA - 三角移动平均

## 描述

三角移动平均（TRIMA）是一个双重平滑的移动平均，将简单移动平均应用于另一个简单移动平均。这创建了一条非常平滑的线，与单一SMA相比减少了噪音和滞后，但比单一SMA有更多滞后。它特别适用于识别长期趋势。

## 分类
重叠研究

## 计算方法

TRIMA通过应用SMA两次来计算：

### 公式
```
TRIMA = SMA(SMA(价格, n), n)
```

其中：
- 第一个SMA = 价格在n个周期内的简单移动平均
- 第二个SMA = 第一个SMA在n个周期内的简单移动平均
- n = 周期（通常两个SMA相同）

### 示例
对于5周期TRIMA：
```
步骤1：计算价格的5周期SMA
步骤2：计算第一个SMA的5周期SMA
结果：TRIMA值
```

## 参数

- **optInTimePeriod**（默认：30）：两个SMA的周期
  - 有效范围：2到100000
  - 常用值：20、30、50

## 输入
- 价格数据：`double[]`（通常是收盘价）

## 输出
- TRIMA值：`double[]`

## 解释

### 特征
- **非常平滑**：双重平滑创建非常平滑的线
- **高滞后**：比单一SMA更多滞后
- **趋势识别**：非常适合长期趋势
- **噪音减少**：过滤短期波动

### 使用
1. **长期趋势**：
   - TRIMA斜率指示趋势方向
   - 价格高于TRIMA = 上升趋势
   - 价格低于TRIMA = 下降趋势

2. **支撑/阻力**：
   - TRIMA在上升趋势中充当动态支撑
   - TRIMA在下降趋势中充当动态阻力

3. **趋势变化**：
   - 价格交叉TRIMA信号主要趋势变化
   - 比快速MA对主要反转更可靠

## 使用示例

```c
// C/C++ 示例
double closePrices[100];
double trimaOutput[100];
int outBegIdx, outNBElement;

// 计算20周期TRIMA
TA_RetCode retCode = TA_TRIMA(
    0,                    // 开始索引
    99,                   // 结束索引
    closePrices,          // 输入价格数据
    20,                   // 时间周期
    &outBegIdx,           // 输出：开始索引
    &outNBElement,        // 输出：元素数量
    trimaOutput           // 输出：TRIMA值
);
```

## 实现细节

TA-Lib TRIMA实现：

1. **第一个SMA**：计算价格的SMA
2. **第二个SMA**：计算第一个SMA的SMA
3. **双重平滑**：创建非常平滑的结果
4. **回望**：第一个输出需要2n-1个周期

## 交易策略

### 1. 长期趋势跟踪
- **买入**：价格交叉TRIMA上方
- **卖出**：价格交叉TRIMA下方
- **持有**：价格在正确一侧时保持头寸
- **最适合**：长期头寸交易

### 2. TRIMA斜率策略
- **买入**：TRIMA斜率为正且价格高于TRIMA
- **卖出**：TRIMA斜率为负且价格低于TRIMA
- **出场**：当斜率变化或价格交叉时
- **最适合**：确认趋势方向

### 3. 多时间框架
- **更高TF**：使用TRIMA确定主要趋势
- **更低TF**：使用快速MA进行入场时机
- **规则**：仅在更高TF TRIMA方向交易
- **最适合**：所有时间框架

### 4. TRIMA + 价格行为
- **上下文**：TRIMA确定趋势
- **入场**：TRIMA触及时的价格模式
- **示例**：上升趋势中牛旗回调到TRIMA
- **确认**：蜡烛图模式
- **最适合**：结合技术分析

## 优势

1. **非常平滑**：出色的噪音减少
2. **趋势清晰**：清晰的趋势识别
3. **可靠**：比快速MA更少假信号
4. **长期焦点**：非常适合头寸交易
5. **简单**：易于理解和使用

## 限制

1. **高滞后**：价格后显著滞后
2. **延迟信号**：错过早期趋势变化
3. **不适合日内交易**：对短期交易太慢
4. **鞭打**：在波动市场中仍可能发生
5. **周期敏感性**：结果因周期选择而异

## 与其他MA的比较

| MA类型 | 平滑性 | 滞后 | 响应性 | 最佳使用 |
|--------|--------|------|--------|----------|
| SMA | 中等 | 中等 | 中等 | 通用 |
| TRIMA | 高 | 高 | 低 | 长期趋势 |
| EMA | 低 | 低 | 高 | 短期 |
| DEMA | 低 | 极低 | 极高 | 日内交易 |
| TEMA | 极低 | 极低 | 极高 | 剥头皮 |

## 相关函数

- **SMA**：简单移动平均 - 构建块
- **EMA**：指数移动平均 - 替代平滑
- **KAMA**：Kaufman自适应移动平均 - 自适应方法
- **MAMA**：MESA自适应移动平均 - 基于周期
- **T3**：T3移动平均 - 平滑低滞后

## 附加说明

三角移动平均得名于双重平滑产生的三角权重模式。虽然它计算为SMA(SMA(价格))，但有效权重模式类似于三角形。

### 关键洞察

1. **双重平滑效果**：
   - 第一个SMA平滑价格数据
   - 第二个SMA平滑第一个SMA
   - 结果：非常平滑、滞后的指标
   - 非常适合主要趋势识别

2. **周期选择**：
   - 更长周期 = 更平滑，更多滞后
   - 更短周期 = 较少平滑，较少滞后
   - 基于交易时间框架平衡
   - 常见：日线图20-50

3. **最佳应用**：
   - 长期趋势识别
   - 头寸交易
   - 过滤市场噪音
   - 确认主要趋势变化

4. **不适合**：
   - 日内交易（太慢）
   - 剥头皮（过度滞后）
   - 短期反转
   - 快速入场/出场策略

5. **组合策略**：
   - 使用TRIMA确定趋势方向
   - 使用快速MA进行入场时机
   - 使用振荡器确定超买/超卖
   - 使用成交量确认

TRIMA对于需要识别主要趋势变化同时过滤短期市场噪音的投资者和头寸交易者特别有价值。它是可用的最平滑移动平均之一，为长期趋势分析提供出色的清晰度。
