# COUNT - 周期内非零值计数

## 描述

COUNT函数计算指定周期内非零值的数量。它是一个简单但有用的数学运算符，可用于各种计算，包括计算有效数据点、识别数据质量，以及作为更复杂指标的构建块。COUNT经常与其他函数结合使用来创建自定义指标。

## 分类
数学运算符

## 计算方法

COUNT计算滚动窗口内非零值的数量：

### 公式
```
COUNT = Σ(1 如果值 ≠ 0, 0 如果值 = 0)
```

其中：
- Σ = 周期上的求和
- 1 = 如果值非零则计数
- 0 = 如果值为零则不计数

### 示例
对于5周期COUNT，值为[10, 0, 12, 0, 9]：
```
COUNT = 1 + 0 + 1 + 0 + 1 = 3
```

## 参数

- **optInTimePeriod**（默认：30）：要计数的周期数
  - 有效范围：2 到 100000
  - 常用值：10、20、50

## 输入
- 价格数据：`double[]`（任何数据）

## 输出
- COUNT 数值：`double[]`（周期内计数）
