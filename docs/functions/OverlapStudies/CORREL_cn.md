# CORREL - 皮尔逊相关系数

## 描述

皮尔逊相关系数（CORREL）测量两个数据系列之间的线性关系。它的范围从-1到+1，其中+1表示完全正相关，-1表示完全负相关，0表示无线性关系。CORREL对于分析不同证券、指标或市场数据之间的关系很有用。

## 分类
统计函数

## 作者
Karl Pearson

## 计算方法

CORREL使用皮尔逊相关公式计算：

### 公式
```
CORREL = Σ[(X - X̄)(Y - Ȳ)] / √[Σ(X - X̄)² × Σ(Y - Ȳ)²]
```

其中：
- X, Y = 数据系列
- X̄, Ȳ = X和Y的均值
- Σ = 周期上的求和

### 替代公式
```
CORREL = 协方差(X,Y) / (σX × σY)
```

其中：
- 协方差(X,Y) = X和Y之间的协方差
- σX, σY = X和Y的标准差

## 参数

- **optInTimePeriod**（默认：30）：计算周期
  - 有效范围：1 到 100000
  - 常用值：20、30、50

## 输入
- **第一个**数据系列：`double[]`
- **第二个**数据系列：`double[]`

## 输出
- 相关数值：`double[]`（范围：-1到+1）

## 解释

### 相关数值
- **+1.0**：完全正相关
- **+0.7到+1.0**：强正相关
- **+0.3到+0.7**：中等正相关
- **-0.3到+0.3**：弱相关
- **-0.7到-0.3**：中等负相关
- **-1.0到-0.7**：强负相关
- **-1.0**：完全负相关

### 交易应用

1. **投资组合分析**：
   - **高正相关**：证券一起移动
   - **高负相关**：证券反向移动
   - **低相关**：独立移动
   - **最适合**：投资组合多样化

2. **指标分析**：
   - **高正相关**：指标一致
   - **高负相关**：指标不一致
   - **低相关**：独立指标
   - **最适合**：指标验证

3. **市场分析**：
   - **高正相关**：市场一起移动
   - **高负相关**：市场反向移动
   - **低相关**：独立市场
   - **最适合**：市场关系分析

## 使用示例

```c
// C/C++ 示例
double series1[100], series2[100];
double correlation[100];
int outBegIdx, outNBElement;

// 计算30周期相关
TA_RetCode retCode = TA_CORREL(
    0,                    // 开始索引
    99,                   // 结束索引
    series1,              // 第一个数据系列
    series2,              // 第二个数据系列
    30,                   // 时间周期
    &outBegIdx,           // 输出：开始索引
    &outNBElement,        // 输出：元素数量
    correlation           // 输出：相关数值
);
```

## 交易策略

### 1. 投资组合多样化策略
- **设置**：计算证券之间的相关
- **入场**：低相关证券
- **出场**：当相关增加时
- **最适合**：投资组合管理

### 2. 指标验证策略
- **设置**：计算指标之间的相关
- **入场**：当指标一致时（高相关）
- **出场**：当指标不一致时（低相关）
- **最适合**：信号确认

### 3. 市场关系策略
- **设置**：计算市场之间的相关
- **入场**：当相关破裂时
- **出场**：当相关重新建立时
- **最适合**：市场分析

### 4. 相关均值回归策略
- **设置**：跟踪相关随时间变化
- **入场**：当相关达到极值时
- **出场**：当相关返回均值时
- **最适合**：相关交易

## 相关分析

### 1. 强正相关（+0.7到+1.0）
- **特征**：证券一起移动
- **交易**：类似交易策略
- **风险**：高相关风险
- **最适合**：趋势跟踪

### 2. 中等正相关（+0.3到+0.7）
- **特征**：某种关系
- **交易**：部分多样化
- **风险**：中等相关风险
- **最适合**：平衡投资组合

### 3. 弱相关（-0.3到+0.3）
- **特征**：独立移动
- **交易**：独立策略
- **风险**：低相关风险
- **最适合**：多样化

### 4. 强负相关（-1.0到-0.7）
- **特征**：反向移动
- **交易**：对冲策略
- **风险**：负相关风险
- **最适合**：对冲

## 优势

1. **客观**：数学关系测量
2. **通用**：适用于任何数据系列
3. **有界**：总是在-1和+1之间
4. **清晰**：易于解释
5. **多功能**：多种应用

## 限制

1. **仅线性**：只测量线性关系
2. **滞后**：基于历史数据
3. **周期依赖**：结果随周期变化
4. **无因果关系**：相关≠因果关系
5. **异常值**：对极值敏感

### 实用技巧

**投资组合分析**：
- 计算证券之间的相关
- 使用低相关进行多样化
- 监控相关变化
- 当相关增加时重新平衡

**指标分析**：
- 计算指标之间的相关
- 使用高相关进行确认
- 避免冗余指标
- 优化指标选择

**市场分析**：
- 计算市场之间的相关
- 用于市场关系分析
- 识别相关破裂
- 用于对冲策略

**风险管理**：
- 监控相关变化
- 用于头寸规模
- 在低相关资产中多样化
- 用负相关资产对冲

CORREL对于投资组合管理、风险分析和理解不同市场数据之间的关系特别有价值。它是定量分析和系统交易策略的重要工具。
