# SUM - 周期内求和

## 描述

SUM函数计算指定周期内值的总和。它是一个简单但有用的数学运算符，可用于各种计算，包括移动总和、累积分析，以及作为更复杂指标的构建块。SUM通常与其他函数结合使用来创建自定义指标。

## 类别
数学运算符

## 计算

SUM计算滚动窗口内值的总和：

### 公式
```
SUM = Σ(价格[i], 价格[i-1], ..., 价格[i-n+1])
```

其中：
- n = 回望周期
- Σ = 周期内的求和

### 示例
对于5周期SUM，价格为[10, 12, 11, 13, 9]：
```
SUM = 10 + 12 + 11 + 13 + 9 = 55
```

## 参数

- **optInTimePeriod**（默认：30）：求和的周期数
  - 有效范围：2到100000
  - 常用值：10, 20, 50

## 输入
- 价格数据：`double[]`（任何价格数据）

## 输出
- SUM值：`double[]`（周期内总和）

TA_RetCode retCode = TA_SUM(

## 实现细节

TA-Lib SUM实现：

1. **滚动窗口**：维护n个值的窗口
2. **求和**：将窗口内所有值相加
3. **高效更新**：为每个新K线高效更新
4. **回望**：在第一个输出之前需要n-1个周期
