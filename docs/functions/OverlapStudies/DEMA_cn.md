# DEMA - 双指数移动平均线

## 描述

双指数移动平均线（DEMA）是由Patrick Mulloy开发的复合指标，试图消除传统移动平均线固有的滞后。尽管其名称，DEMA不是简单的EMA的EMA。相反，它使用单EMA和双EMA的组合来创建响应更快的移动平均线。

## 分类
重叠研究

## 作者
Patrick Mulloy

## 计算方法

DEMA使用单EMA和双平滑EMA之间的差值来减少滞后：

### 公式
```
DEMA = 2 × EMA(n) - EMA(EMA(n))
```

其中：
- EMA(n) = 周期n的指数移动平均线
- EMA(EMA(n)) = 应用于第一个EMA的EMA

### 详细步骤
```
步骤1：计算价格的EMA
        EMA1 = EMA(价格, 周期)

步骤2：计算EMA1的EMA
        EMA2 = EMA(EMA1, 周期)

步骤3：计算DEMA
        DEMA = (2 × EMA1) - EMA2
```

减法有效地补偿了双EMA中的滞后。

## 参数

- **optInTimePeriod**（默认：30）：指数移动平均线的周期
  - 有效范围：2 到 100000
  - 常用值：9、20、50

## 输入
- 价格数据：`double[]`（通常是收盘价）

## 输出
- DEMA 数值：`double[]`

## 解释

### 响应性
- **比SMA/EMA更敏感**：对价格变化反应更快
- **更少滞后**：与传统移动平均线相比减少滞后
- **趋势识别**：更好地跟踪价格

### 使用方法
像任何移动平均线一样使用：
1. **趋势方向**：
   - 价格高于DEMA = 上升趋势
   - 价格低于DEMA = 下降趋势
   - DEMA斜率表示趋势强度

2. **交叉**：
   - 价格向上穿越DEMA = 买入信号
   - 价格向下穿越DEMA = 卖出信号

3. **支撑/阻力**：
   - DEMA在上升趋势中充当动态支撑
   - DEMA在下降趋势中充当动态阻力

4. **多DEMA系统**：
   - 使用两个DEMA（例如，9和20周期）
   - 快速DEMA穿越慢速 = 趋势变化信号

## 实现细节

TA-Lib DEMA 实现：

1. **第一个EMA**：计算价格的EMA
2. **第二个EMA**：计算第一个EMA的EMA
3. **组合**：应用DEMA公式：2×EMA1 - EMA2
4. **回望**：由于双平滑，比单EMA更长

## 滞后减少解释

### 为什么DEMA滞后更少

EMA的双重应用实际上增加了滞后，但通过从2倍单EMA中减去它，额外的滞后被移除，然后更多：

```
单EMA：     有滞后 = L
双EMA：     有滞后 ≈ 2L
DEMA公式：   2×EMA - EMA(EMA) 移除额外滞后
结果：       有效滞后 < L
```

这个巧妙的数学技巧产生了一个更紧密跟随价格的移动平均线。

## 交易策略

### 1. DEMA交叉策略
- **买入**：价格向上穿越DEMA
- **卖出**：价格向下穿越DEMA
- **优势**：比SMA/EMA更早的信号
- **最适合**：趋势市场

### 2. 双DEMA策略
- **设置**：使用两个DEMA（例如，9和21周期）
- **买入**：快速DEMA向上穿越慢速DEMA
- **卖出**：快速DEMA向下穿越慢速DEMA
- **最适合**：早期捕捉趋势变化

### 3. DEMA + RSI策略
- **趋势**：由DEMA确定
- **入场**：在上升趋势中RSI超卖（DEMA上升）
- **出场**：价格向下穿越DEMA或RSI超买
- **最适合**：趋势中的回调入场

### 4. DEMA带状
- **设置**：多个DEMA（例如，5、10、15、20、25、30）
- **上升趋势**：带状扩张，价格高于所有DEMA
- **下降趋势**：带状扩张，价格低于所有DEMA
- **横盘**：带状收缩
- **最适合**：视觉趋势评估

## 优势

1. **减少滞后**：比SMA或EMA显著减少滞后
2. **更敏感**：对价格变化反应更快
3. **比简单MA更平滑**：比仅仅减少周期更好
4. **趋势跟踪**：非常适合趋势识别
5. **通用**：适用于所有市场和时间框架

## 限制和考虑

1. **更多震荡**：增加的响应性意味着更多假信号
2. **仍然滞后**：虽然减少，但滞后并未完全消除
3. **震荡市场**：在区间中可能产生过多信号
4. **超调**：在急剧反转期间可能超调
5. **复杂性**：比简单移动平均线更复杂

## DEMA vs. 其他移动平均线

| MA类型 | 滞后 | 响应性 | 平滑性 | 复杂性 |
|--------|------|--------|--------|--------|
| SMA | 高 | 低 | 高 | 低 |
| EMA | 中等 | 中等 | 中等 | 低 |
| WMA | 中等 | 中等 | 中等 | 低 |
| DEMA | 低 | 高 | 低 | 中等 |
| TEMA | 极低 | 极高 | 极低 | 高 |

DEMA在响应性和复杂性方面位于EMA和TEMA之间。

## 相关函数

- **EMA**：指数移动平均线 - DEMA的基础
- **TEMA**：三指数移动平均线 - 甚至更少滞后
- **KAMA**：Kaufman自适应移动平均线 - 自适应方法
- **T3**：T3移动平均线 - 低滞后平滑
- **SMA**：简单移动平均线 - 基线比较

## 附加说明

Patrick Mulloy在1990年代开发了DEMA来解决所有移动平均线固有的滞后问题。他的关键洞察是通过使用单EMA和双EMA之间的差值，他可以创建一个比单独组件响应更快的复合指标。

### 关键洞察

1. **不仅仅是平滑两次**：
   - DEMA ≠ EMA(EMA(价格))
   - DEMA = 2×EMA(价格) - EMA(EMA(价格))
   - 减法对滞后减少至关重要

2. **数学优雅**：
   - 使用指数平滑的特性
   - 巧妙地抵消额外滞后
   - 在不牺牲太多平滑性的情况下产生更快响应

3. **实际应用**：
   - 最好在趋势市场中使用
   - 产生更早的入场信号
   - 由于更多震荡可能需要更宽的止损
   - 考虑使用成交量确认

4. **周期选择**：
   - 较短周期（9-15）：日内交易，非常敏感
   - 中等周期（20-30）：摆动交易，平衡
   - 较长周期（50-100）：头寸交易，更平滑

5. **与TEMA比较**：
   - TEMA进一步发展概念（三重平滑）
   - DEMA提供速度与可靠性的良好平衡
   - TEMA更快但更容易震荡
   - 根据交易风格和市场条件选择

DEMA在需要比传统移动平均线提供更快信号但希望比简单使用较短周期EMA更可靠的日内交易者和摆动交易者中特别受欢迎。
