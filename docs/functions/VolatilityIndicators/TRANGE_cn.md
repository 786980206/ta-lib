# TRANGE - 真实范围

## 描述

真实范围（TRANGE）是一个波动性指标，测量单个周期内价格移动的真实范围。它是平均真实范围（ATR）指标的构建块，提供对价格移动波动性的洞察。TRANGE考虑周期之间的缺口，提供比简单高低范围更准确的波动性测量。

## 分类
波动性指标

## 作者
J. Welles Wilder

## 计算方法

TRANGE计算为三个值的最大值：

### 公式
```
TRANGE = max(最高价 - 最低价, |最高价 - 前一个收盘价|, |最低价 - 前一个收盘价|)
```

其中：
- 最高价 = 当前周期最高价
- 最低价 = 当前周期最低价
- 前一个收盘价 = 前一个周期收盘价
- | | = 绝对值

### 示例
对于周期：
- 最高价 = 105
- 最低价 = 100
- 前一个收盘价 = 102

```
TRANGE = max(105 - 100, |105 - 102|, |100 - 102|)
       = max(5, 3, 2)
       = 5
```

## 参数

无 - TRANGE为每个周期计算

## 输入
- **最高价**：`double[]`
- **最低价**：`double[]`
- **收盘价**：`double[]`

## 输出
- 真实范围值：`double[]`（始终为正）

## 解释

### TRANGE值
- **高TRANGE**：高波动性周期
- **低TRANGE**：低波动性周期
- **零TRANGE**：无价格移动（罕见）
- **增加TRANGE**：波动性增加
- **减少TRANGE**：波动性减少

### 交易应用

1. **波动性分析**：
   - **高TRANGE**：高波动性周期
   - **低TRANGE**：低波动性周期
   - **TRANGE趋势**：波动性趋势分析
   - **最适合**：基于波动性的策略

2. **风险评估**：
   - **高TRANGE**：高风险周期
   - **低TRANGE**：低风险周期
   - **TRANGE模式**：风险模式分析
   - **最适合**：风险管理

3. **市场分析**：
   - **高TRANGE**：不稳定市场
   - **低TRANGE**：稳定市场
   - **TRANGE模式**：市场状态识别
   - **最适合**：市场分析

## 使用示例

```c
// C/C++ 示例
double high[100], low[100], close[100];
double trangeOutput[100];
int outBegIdx, outNBElement;

// 计算真实范围
TA_RetCode retCode = TA_TRANGE(
    0,                    // 开始索引
    99,                   // 结束索引
    high,                 // 最高价
    low,                  // 最低价
    close,                // 收盘价
    &outBegIdx,           // 输出：开始索引
    &outNBElement,        // 输出：元素数量
    trangeOutput          // 输出：真实范围值
);
```

## 实现细节

TA-Lib TRANGE实现：

1. **缺口考虑**：考虑周期之间的缺口
2. **最大值计算**：找到三个值的最大值
3. **绝对值**：对缺口使用绝对值
4. **回望**：第一个输出需要1个周期

## 交易策略

### 1. 基于波动性的策略
- **设置**：跟踪TRANGE随时间变化
- **入场**：当TRANGE达到极值时
- **出场**：当TRANGE回到均值时
- **最适合**：波动性交易

### 2. 风险管理策略
- **设置**：使用TRANGE进行头寸规模
- **入场**：在高TRANGE期间减少规模
- **出场**：在低TRANGE期间增加规模
- **最适合**：风险管理

### 3. 市场状态策略
- **设置**：识别TRANGE状态
- **入场**：不同状态使用不同策略
- **出场**：状态变化时切换策略
- **最适合**：自适应交易

### 4. TRANGE均值回归策略
- **设置**：跟踪TRANGE均值回归
- **入场**：当TRANGE偏离均值时
- **出场**：当TRANGE回到均值时
- **最适合**：波动性交易

## TRANGE vs. 高低范围

| 方面 | TRANGE | 高低范围 |
|------|--------|----------|
| 缺口 | 考虑缺口 | 忽略缺口 |
| 准确性 | 更准确 | 较不准确 |
| 波动性 | 更好的波动性测量 | 有限的波动性测量 |
| 最适合 | 波动性分析 | 简单范围分析 |

## 优势

1. **准确**：比简单范围更准确
2. **缺口考虑**：考虑周期之间的缺口
3. **通用**：适用于所有市场
4. **简单**：易于理解和计算
5. **多功能**：许多应用

## 限制

1. **单周期**：仅测量一个周期
2. **无平滑**：原始波动性测量
3. **噪音**：可能非常噪音
4. **上下文依赖**：需要解释
5. **无方向**：不指示方向

## 相关函数

- **NATR**：标准化平均真实范围 - 百分比版本
- **STDDEV**：标准差 - 替代波动性测量
- **VAR**：方差 - 替代波动性测量

### 关键洞察

1. **缺口考虑**：

   - 更适合缺口分析

2. **波动性测量**：
   - 测量真实价格波动性
   - 更高TRANGE = 更高波动性
   - 更低TRANGE = 更低波动性
   - 始终正值

3. **最佳应用**：
   - 波动性分析
   - 风险评估
   - 市场状态识别
   - ATR计算

4. **交易应用**：
   - 基于波动性的策略
   - 风险管理
   - 头寸规模

5. **组合策略**：
