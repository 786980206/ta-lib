# STDDEV - 标准差

## 描述

标准差是一个统计测量，量化数据集中变化或分散的数量。在技术分析中，它测量价格波动性 - 价格偏离其平均值的程度。高标准差表示高波动性（大价格摆动），而低标准差表示低波动性（稳定价格）。

## 分类
统计函数/波动性指标

## 计算方法

标准差测量数据点与均值的平均距离：

### 公式（总体标准差）
```
STDDEV = sqrt(Σ(价格 - 均值)² / n)
```

其中：
- 均值 = n个周期内的平均价格
- n = 周期数
- sqrt = 平方根

### 步骤
```
步骤1：计算价格的均值（平均值）
步骤2：对于每个价格，计算与均值的平方差
步骤3：平均平方差
步骤4：取结果的平方根
```

### 示例
对于价格[10, 12, 14, 16, 18]：
```
均值 = (10+12+14+16+18)/5 = 14
平方差：16, 4, 0, 4, 16
平方差的平均值：40/5 = 8
STDDEV = sqrt(8) = 2.83
```

## 参数

- **optInTimePeriod**（默认：5）：周期数
  - 有效范围：2到100000
  - 常用值：10、20、30

- **optInNbDev**（默认：1.0）：偏差数（乘数）
  - 有效范围：-3.0e+37到3.0e+37
  - 常用值：1.0、2.0、3.0

## 输入
- 价格数据：`double[]`（通常是收盘价）

## 输出
- 标准差值：`double[]`（与价格相同单位）

## 解释

### 波动性评估
- **高STDDEV**：高波动性，大价格摆动，风险高
- **低STDDEV**：低波动性，稳定价格，风险较低
- **上升STDDEV**：波动性增加
- **下降STDDEV**：波动性减少

### 应用
1. **风险测量**：更高STDDEV = 更高风险
2. **波动性分析**：跟踪波动性变化
3. **布林带**：STDDEV定义带宽度
4. **头寸规模**：根据波动性调整规模
5. **止损**：使用STDDEV倍数进行止损

## 使用示例

```c
// C/C++ 示例
double closePrices[100];
double stddevOutput[100];
int outBegIdx, outNBElement;

// 计算20周期标准差
TA_RetCode retCode = TA_STDDEV(
    0,                    // 开始索引
    99,                   // 结束索引
    closePrices,          // 输入价格数据
    20,                   // 时间周期
    1.0,                  // 偏差数
    &outBegIdx,           // 输出：开始索引
    &outNBElement,        // 输出：元素数量
    stddevOutput          // 输出：STDDEV值
);
```

## 实现细节

TA-Lib STDDEV实现：

1. **均值计算**：计算周期内的平均值
2. **方差计算**：计算平均平方偏差
3. **标准差**：取方差的平方根
4. **乘数**：应用nbDev参数
5. **滚动窗口**：为每个柱高效更新

## 交易应用

### 1. 基于波动性的头寸规模
```
头寸规模 = 风险金额 / (入场 - 止损)
止损距离 = 入场 ± (STDDEV × 乘数)
```

根据波动性调整头寸。

### 2. 波动性突破
- **设置**：STDDEV收缩到低水平
- **信号**：价格突破且STDDEV扩张
- **入场**：在STDDEV上升时突破
- **最适合**：在盘整期后

### 3. 布林带应用
布林带使用STDDEV：
```
上带 = SMA + (2 × STDDEV)
下带 = SMA - (2 × STDDEV)
```

### 4. 风险调整止损
```
止损 = 入场 ± (2 × STDDEV)
```

适应当前波动性。

## 统计显著性

### 正态分布
如果价格遵循正态分布：
- **±1 STDDEV**：包含约68%的价格数据
- **±2 STDDEV**：包含约95%的价格数据
- **±3 STDDEV**：包含约99.7%的价格数据

超过2-3个标准差的移动在统计上是罕见的。

## 优势

1. **客观**：清晰的数学测量
2. **波动性量化**：精确的波动性测量
3. **通用**：适用于所有市场
4. **风险评估**：直接的风险测量
5. **基础**：许多其他指标的基础

## 限制

1. **滞后**：基于历史数据
2. **假设正态分布**：市场不总是正态的
3. **无方向**：不指示趋势方向
4. **异常值**：对极值敏感
5. **周期依赖**：结果因周期而异

## 相关函数

- **VAR**：方差 - STDDEV的平方
- **BBANDS**：布林带 - 使用STDDEV
- **ATR**：平均真实范围 - 替代波动性测量
- **NATR**：标准化ATR - 基于百分比的波动性

## 附加说明

标准差是交易和风险管理中最重要的统计测量之一。它是以下内容的基础：
- 布林带
- 夏普比率
- 风险价值（VaR）
- 投资组合优化
- 风险调整回报

John Bollinger通过布林带在技术分析中推广了它的使用，但STDDEV本身对于以下内容很有用：
- 比较跨证券的波动性
- 基于波动性周期时机入场/出场
- 设置动态止损和目标
- 调整头寸规模
- 评估市场条件
