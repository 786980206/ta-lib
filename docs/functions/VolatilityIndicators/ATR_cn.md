# ATR - 平均真实区间

## 描述

平均真实区间（ATR）是一个波动性指标，通过计算指定周期内真实区间的平均值来衡量价格波动性的程度。与许多指标不同，ATR不提供价格方向或趋势的指示；相反，它测量市场的波动性水平。

## 分类
波动性指标

## 作者
J. Welles Wilder

## 计算方法

ATR的计算分为三个步骤：

### 步骤1：计算真实区间（TR）
真实区间是以下三个值中的最大值：
1. 当前最高价 - 当前最低价
2. |当前最高价 - 前收盘价|
3. |当前最低价 - 前收盘价|

```
TR = max[(H - L), |H - Cp|, |L - Cp|]
```

其中：
- H = 当前最高价
- L = 当前最低价
- Cp = 前收盘价
- | | = 绝对值

### 步骤2：计算初始ATR
```
初始ATR = 前n个真实区间的平均值
```

### 步骤3：计算后续ATR（Wilder平滑）
```
ATR = ((前一个ATR × (n-1)) + 当前TR) / n
```

这是Wilder的平滑方法（类似于指数移动平均线但不同）。

## 参数

- **optInTimePeriod**（默认：14）：ATR计算的周期数
  - 有效范围：1 到 100000
  - 常用值：7、14、21

## 输入
- **最高价**：`double[]`
- **最低价**：`double[]`
- **收盘价**：`double[]`

## 输出
- ATR 数值：`double[]`（与价格相同的单位）

## 解释

### 波动性评估
- **高ATR**：高波动性，大价格摆动
- **低ATR**：低波动性，小价格摆动
- **上升ATR**：波动性增加
- **下降ATR**：波动性减少

### 关键概念

1. **绝对测量**：ATR以价格单位表示，不是百分比
   - 50美元股票ATR为3美元 = 6%波动性
   - 100美元股票ATR为3美元 = 3%波动性
   - 不能直接比较不同价格证券的ATR值

2. **非方向性**：ATR不指示趋势方向
   - 上升趋势中的高ATR = 强买入
   - 下降趋势中的高ATR = 强卖出
   - 低ATR = 盘整或弱趋势

3. **市场特征**：
   - 趋势市场通常有更高的ATR
   - 横盘市场通常有更低的ATR
   - 突破通常伴随着上升的ATR

## 使用示例

```c
// C/C++ 示例
double high[100], low[100], close[100];
double atrOutput[100];
int outBegIdx, outNBElement;

// 计算14周期ATR
TA_RetCode retCode = TA_ATR(
    0,                    // 开始索引
    99,                   // 结束索引
    high,                 // 最高价
    low,                  // 最低价
    close,                // 收盘价
    14,                   // 时间周期
    &outBegIdx,           // 输出：开始索引
    &outNBElement,        // 输出：元素数量
    atrOutput             // 输出：ATR数值
);
```

## 实现细节

TA-Lib ATR 实现：

1. **真实区间计算**：特别处理第一个柱（无前收盘价）
2. **初始平均值**：对第一个ATR值使用简单平均值
3. **Wilder平滑**：对后续值应用类似指数的平滑
4. **跳空处理**：真实区间捕获会话间的跳空
5. **回望周期**：需要`timePeriod`个柱才能产生第一个输出

### 计算步骤
```
步骤1：对每个柱，计算TR = max(H-L, |H-Cp|, |L-Cp|)
步骤2：初始ATR = 前n个TR的平均值
步骤3：后续ATR = ((前一个ATR × (n-1)) + 当前TR) / n
```

## 交易应用

### 1. 止损放置
ATR最常见的用途：

```
止损距离 = ATR × 乘数

多头头寸：
止损 = 入场价 - (ATR × 2)

空头头寸：
止损 = 入场价 + (ATR × 2)
```

常用乘数：1.5、2、2.5、3

**优势**：
- 适应市场波动性
- 在波动市场中更宽的止损（避免过早止损）
- 在平静市场中更紧的止损（保护资本）

### 2. 头寸规模
使用ATR进行基于风险的头寸规模：

```
头寸规模 = (账户风险$) / (ATR × 乘数)
```

示例：
- 账户：100,000美元
- 每笔交易风险：1% = 1,000美元
- ATR：2.50美元
- 乘数：2

头寸规模 = 1,000美元 / (2.50美元 × 2) = 200股

### 3. 利润目标
基于ATR设置利润目标：

```
目标 = 入场价 + (ATR × 倍数)
```

常用倍数：2、3、4（目标2:1或3:1的回报:风险比）

### 4. 突破验证
用ATR确认突破：

- **有效突破**：ATR扩大到最近平均值之上
- **假突破**：ATR保持低位
- 上升ATR确认真实移动

### 5. 吊灯出场
基于ATR的跟踪止损：

```
多头吊灯出场 = 最高价 - (ATR × 乘数)
空头吊灯出场 = 最低价 + (ATR × 乘数)
```

跟随价格同时为正常波动性留出空间。

## 波动性分析

### ATR模式

1. **波动性收缩**：
   - ATR下降到多周期低点
   - 表示盘整
   - 通常先于重大移动
   - 方向未知直到突破

2. **波动性扩张**：
   - ATR急剧上升
   - 表示强移动正在进行
   - 确认突破和趋势
   - 动量建立

3. **波动性周期**：
   - 市场在高和低波动性之间交替
   - 扩张后是收缩
   - 收缩后是扩张
   - 用于时机和期望设置

### 历史ATR分析

将当前ATR与历史水平比较：

```
ATR百分位数 = 当前ATR在过去X个周期中的排名
```

- **高百分位数（>80%）**：极高波动性
- **低百分位数（<20%）**：极低波动性
- 用于背景和均值回归期望

## 周期选择

### 不同周期
- **7周期**：短期，更敏感，适合日内交易者
- **14周期**：标准，Wilder推荐，最常见
- **21周期**：长期，更平滑，适合头寸交易者
- **50周期**：非常长期，显示主要波动性变化

### 时间框架考虑
- **日内**：较短周期（7-10）
- **日线**：标准周期（14）
- **周线**：可以使用标准或稍长（14-21）

## 优势

1. **客观**：清晰、可量化的波动性测量
2. **自适应**：自动适应市场条件
3. **通用**：适用于所有市场和时间框架
4. **风险管理**：非常适合止损和头寸规模
5. **简单**：易于理解和计算
6. **实用**：直接应用于交易决策

## 限制和考虑

1. **非方向性**：不指示趋势方向
2. **绝对值**：不能比较不同价格证券
3. **滞后**：像所有平均值一样，滞后当前波动性
4. **无价格水平**：不指示支撑/阻力
5. **平滑**：Wilder平滑可能延迟对波动性变化的响应
6. **周期敏感性**：不同周期给出不同读数

## 相关函数

- **NATR**：标准化平均真实区间 - ATR的百分比版本
- **TRANGE**：真实区间 - 单周期，不平均
- **STDDEV**：标准差 - 另一个波动性测量
- **BBANDS**：布林带 - 使用标准差
- **KELTNER**：肯特纳通道 - 使用ATR作为带宽

## 附加说明

J. Welles Wilder在1978年的《技术交易系统新概念》一书中介绍了平均真实区间。它最初是为商品交易开发的，但已证明在所有市场中都有效。

### 关键洞察

1. **真实区间创新**：Wilder的真实区间捕获：
   - 正常日内波动性（H - L）
   - 跳空上涨波动性（H - Cp）
   - 跳空下跌波动性（Cp - L）
   - 最全面的单周期波动性测量

2. **Wilder平滑**：不同于SMA或EMA：
   - 对历史值给予更多权重
   - 比同等周期EMA更平滑
   - 不太容易产生不规则移动
   - 更适合趋势跟踪应用

3. **风险管理黄金标准**：
   - 专业交易者广泛使用ATR进行：
     * 止损放置
     * 头寸规模
     * 风险评估
     * 投资组合波动性管理

4. **市场比较**：
   - 比较证券间波动性：
   - 使用NATR（标准化ATR）
   - 或计算：ATR / 价格 × 100

5. **波动性与价格方向**：
   - 高ATR可以出现在任何趋势中：
     * 强上升趋势 = 高ATR
     * 强下降趋势 = 高ATR
     * 波动性横盘 = 高ATR
   - 与趋势指标一起使用以获得背景

6. **吊灯出场**：
   - 最有效的基于ATR的系统之一
   - 由Chuck LeBeau创建
   - 跟踪止损在：最高价 - (3 × ATR)
   - 在保护利润的同时给予足够空间

### 实用技巧

- **多ATR策略**：使用2× ATR进行止损，4× ATR进行目标
- **波动性过滤器**：只有当ATR高于平均值时才交易（高概率设置）
- **入场时机**：在入场前等待ATR在收缩后扩张
- **避免低波动性**：当ATR处于多月低点时，减少头寸规模或等待

ATR是专业交易中最实用和广泛使用的指标之一，主要是因为它直接转化为风险管理和头寸规模决策，而不仅仅是提供信号。
