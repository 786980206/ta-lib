# NATR - 标准化平均真实范围

## 描述

标准化平均真实范围（NATR）是一个波动性指标，测量平均真实范围作为当前价格的百分比。它类似于ATR但表示为百分比，使得更容易比较不同证券和价格水平的波动性。NATR有助于识别相对波动性并提供价格波动性的标准化测量。

## 分类
波动性指标

## 作者
J. Welles Wilder（改编）

## 计算方法

NATR计算为ATR相对于当前价格的百分比：

### 公式
```
NATR = (ATR / 收盘价) × 100
```

其中：
- ATR = 平均真实范围
- 收盘价 = 当前收盘价
- 结果表示为百分比

### 示例
如果ATR = 2.5，收盘价 = 100：
```
NATR = (2.5 / 100) × 100 = 2.5%
```

## 参数

- **optInTimePeriod**（默认：14）：ATR计算的周期
  - 有效范围：1到100000
  - 常用值：14（标准）、20、30

## 输入
- **最高价**：`double[]`
- **最低价**：`double[]`
- **收盘价**：`double[]`

## 输出
- NATR值：`double[]`（百分比）

## 解释

### NATR值
- **高NATR**：高相对波动性
- **低NATR**：低相对波动性
- **零NATR**：无波动性（罕见）
- **增加NATR**：波动性增加
- **减少NATR**：波动性减少

### 交易应用

1. **波动性分析**：
   - **高NATR**：高波动性期间
   - **低NATR**：低波动性期间
   - **NATR趋势**：波动性趋势分析
   - **最适合**：基于波动性的策略

2. **风险评估**：
   - **高NATR**：高风险期间
   - **低NATR**：低风险期间
   - **NATR模式**：风险模式分析
   - **最适合**：风险管理

3. **市场分析**：
   - **高NATR**：不稳定市场
   - **低NATR**：稳定市场
   - **NATR模式**：市场状态识别
   - **最适合**：市场分析

## 使用示例

```c
// C/C++ 示例
```

## 交易策略

### 1. 基于波动性的策略
- **设置**：跟踪NATR随时间变化
- **入场**：当NATR达到极值时
- **出场**：当NATR回到均值时
- **最适合**：波动性交易

### 2. 风险管理策略
- **设置**：使用NATR进行头寸规模
- **入场**：在高NATR期间减少规模
- **出场**：在低NATR期间增加规模
- **最适合**：风险管理

### 3. 市场状态策略
- **设置**：识别NATR状态
- **入场**：不同状态使用不同策略
- **出场**：状态变化时切换策略
- **最适合**：自适应交易

### 4. NATR均值回归策略
- **设置**：跟踪NATR均值回归
- **入场**：当NATR偏离均值时
- **出场**：当NATR回到均值时
- **最适合**：波动性交易

## NATR vs. ATR

| 方面 | NATR | ATR |
|------|------|-----|
| 单位 | 百分比 | 绝对值 |
| 比较 | 跨证券容易 | 跨证券困难 |
| 缩放 | 自动 | 手动 |
| 解释 | 相对波动性 | 绝对波动性 |
| 最适合 | 跨证券分析 | 单一证券 |

## 优势

1. **标准化**：百分比使比较容易
2. **通用**：适用于所有证券
3. **可缩放**：无需手动缩放
4. **清晰**：易于理解的百分比
5. **投资组合**：对投资组合分析有用

## 限制

1. **仍有滞后**：基于历史数据
2. **鞭打**：在波动市场中可能发生
3. **周期敏感性**：结果因周期而异
4. **无方向**：不指示方向
5. **上下文依赖**：需要解释

## 周期选择

### 短周期（10-14）
- **特征**：更具响应性
- **使用**：短期交易
- **权衡**：更多信号，更多噪音
- **最适合**：主动交易

### 标准周期（14-20）
- **特征**：平衡方法
- **使用**：一般波动性分析
- **权衡**：良好平衡
- **最适合**：大多数应用

### 长周期（20-30）
- **特征**：更平滑，响应性较低
- **使用**：长期分析
- **权衡**：较少信号，更可靠
- **最适合**：头寸交易

## 附加说明

NATR是作为ATR的改进而开发的，通过使用百分比而不是绝对值，使得更容易比较不同证券的波动性。

### 关键洞察

1. **百分比优势**：
   - 容易跨证券比较
   - 无需手动缩放
   - 通用解释
   - 投资组合分析友好

2. **波动性测量**：
   - 测量相对波动性
   - 更高NATR = 更高波动性
   - 更低NATR = 更低波动性
   - 总是正值

3. **最佳应用**：
   - 跨证券波动性分析
   - 投资组合波动性评估
   - 风险管理
   - 市场状态识别

4. **交易应用**：
   - 基于波动性的策略
   - 风险管理
   - 头寸规模
   - 止损放置

5. **组合策略**：
   - 与趋势指标一起使用
   - 与成交量分析结合
   - 用于波动性分析
   - 多时间框架分析

### 实用技巧

**对于波动性分析**：
- 跟踪NATR随时间变化
- 识别波动性状态
- 用于波动性交易
- 监控波动性变化

**对于风险管理**：
- 使用NATR进行头寸规模
- 在高NATR期间减少规模
- 在低NATR期间增加规模
- 监控风险趋势

**对于市场分析**：
- 识别市场状态
- 使用NATR进行市场分类
- 监控NATR模式
- 用于市场时机

**对于投资组合分析**：
- 比较跨证券的NATR
- 按波动性排序
- 按波动性多样化
- 用于资产配置
