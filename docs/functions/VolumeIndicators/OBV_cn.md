# OBV - 平衡成交量

## 描述

平衡成交量（OBV）是一个动量指标，使用成交量流来预测股价变化。它通过在上涨日添加成交量，在下跌日减去成交量，将成交量与价格变化联系起来，创建累积总量。OBV基于成交量先于价格移动的前提。

## 分类
成交量指标

## 作者
Joseph Granville

## 计算方法

OBV使用简单的累积公式计算：

```
如果收盘价 > 前一个收盘价：
    OBV = 前一个OBV + 成交量

如果收盘价 < 前一个收盘价：
    OBV = 前一个OBV - 成交量

如果收盘价 = 前一个收盘价：
    OBV = 前一个OBV
```

计算从任意值开始（通常是零或第一天的成交量），然后从那里构建。

### 公式
```
OBV(t) = OBV(t-1) + sign(收盘价(t) - 收盘价(t-1)) × 成交量(t)
```

其中：
- sign = 如果收盘价更高则为+1，如果更低则为-1，如果不变则为0

## 参数

无 - OBV仅使用价格和成交量数据

## 输入
- **收盘价**：`double[]`
- **成交量**：`double[]`

## 输出
- OBV值：`double[]`（累积，无界）

## 解释

### 基本原理
- **成交量领先价格**：OBV的变化通常先于价格变化
- **上升OBV**：累积，买入压力
- **下降OBV**：派发，卖出压力

### 交易信号

1. **趋势确认**：
   - **看涨**：价格上涨 + OBV上涨（健康上升趋势）
   - **看跌**：价格下跌 + OBV下跌（健康下降趋势）
   - 两者一起趋势确认趋势强度

2. **背离**（最重要）：
   - **看涨背离**：
     * 价格创更低低点
     * OBV创更高低点
     * 建议累积，潜在向上反转
     * 比单独价格更可靠

   - **看跌背离**：
     * 价格创更高高点
     * OBV创更低高点
     * 建议派发，潜在向下反转
     * 多头头寸的警告信号

3. **突破**：
   - **确认突破**：OBV与价格一起突破阻力
   - **假突破**：OBV不确认价格突破
   - OBV确认大大增加可靠性

4. **OBV趋势线**：
   - 在OBV上绘制趋势线，就像在价格上一样
   - OBV趋势线突破通常先于价格突破
   - 可以提供早期警告信号

## 使用示例

```c
// C/C++ 示例
double close[100], volume[100];
double obvOutput[100];
int outBegIdx, outNBElement;

// 计算OBV
TA_RetCode retCode = TA_OBV(

    99,                   // 结束索引

    volume,               // 成交量
    &outBegIdx,           // 输出：开始索引
    &outNBElement,        // 输出：元素数量
    obvOutput             // 输出：OBV值
);
```

## 实现细节

TA-Lib OBV实现：

1. **简单累积**：根据价格方向添加或减去成交量
2. **初始值**：从零或第一个成交量值开始
3. **不变价格**：价格不变时不添加/减去成交量
4. **无平滑**：原始累积值，无平均
5. **快速计算**：非常高效，O(n)复杂度

### 计算步骤
```
步骤1：比较今天收盘价与昨天收盘价
步骤2：如果更高，将今天成交量添加到OBV
        如果更低，从OBV减去今天成交量
        如果相等，OBV保持不变
步骤3：对所有柱重复
```

## 交易策略

### 1. 背离策略
- **设置**：识别价格和OBV之间的背离
- **入场**：等待价格确认（反转模式）
- **止损**：在最近摆动高点/低点外
- **目标**：前一个摆动或测量移动
- **最适合**：识别趋势反转

### 2. 突破确认
- **设置**：价格接近阻力/支撑
- **确认**：OBV也接近突破水平
- **入场**：当两者同时突破时
- **止损**：突破水平下方/上方
- **目标**：模式的测量移动
- **最适合**：区间突破

### 3. 趋势跟踪
- **趋势**：用MA或ADX建立趋势
- **入场**：当OBV确认趋势时的回调
- **过滤器**：仅当OBV与价格趋势时交易
- **出场**：当OBV突破趋势时
- **最适合**：强趋势市场

### 4. 相对强度
- **方法**：比较股票OBV与指数OBV
- **强股票**：股票OBV比指数上升更快
- **弱股票**：股票OBV上升较慢或下跌
- **使用**：用于多头投资组合中的股票选择

## OBV分析技术

### 1. OBV移动平均线
对OBV应用移动平均线：
```
OBV MA = SMA(OBV, 周期)
```

信号：
- **买入**：OBV交叉其MA上方
- **卖出**：OBV交叉其MA下方
- 平滑OBV噪音

### 2. OBV变化率
```
OBV ROC = (OBV - OBV n个周期前) / OBV n个周期前 × 100
```

- 显示成交量累积动量
- 有助于识别加速/减速

### 3. OBV振荡器
```
OBV振荡器 = ((OBV - OBV MA) / OBV MA) × 100
```

- 标准化OBV用于比较
- 在零周围振荡
- 更容易识别极值

### 4. OBV趋势线
- 在OBV图表上绘制趋势线
- 突破通常先于价格趋势线突破
- 用于早期警告系统

## 优势

1. **简单**：易于理解和计算
2. **领先指标**：可以先于价格变化
3. **成交量整合**：包含重要成交量数据
4. **趋势确认**：验证价格移动
5. **背离检测**：有效发现背离
6. **通用**：适用于所有市场和时间框架

## 限制和考虑

1. **绝对值无意义**：只有方向和背离重要
2. **无标准化**：无法比较不同股票的OBV
3. **二元逻辑**：全有或全无的成交量分配（比A/D线更不细致）
4. **缺口处理**：不区分小和大价格变化
5. **无超买/超卖**：无定义的极值水平
6. **需要确认**：不应作为独立指标使用

## OBV vs. A/D线

| 特征 | OBV | A/D线 |
|------|-----|-------|
| 成交量分配 | 二元（全部进入/退出） | 比例（基于收盘位置） |
| 复杂度 | 更简单 | 更复杂 |
| 价格敏感性 | 收盘价与前一个收盘价 | 收盘价与高低范围 |
| 开发者 | Joseph Granville | Marc Chaikin |
| 最适合 | 趋势确认 | 精细分析 |

两者都有用；许多交易者一起使用两者。

## 市场应用

### 不同市场条件

**趋势市场**：
- OBV最有效
- 强趋势显示持续的OBV方向
- 背离更有意义
- 用于趋势确认

**区间市场**：
- 效果较差
- 可能产生假信号
- 最好使用振荡器
- 等待突破

**波动市场**：
- 观察背离
- 可以信号转折点
- 与价格模式结合
- 假信号率较高

**低成交量市场**：
- 可靠性较低
- 成交量数据意义较小
- 考虑其他指标
- 对信号谨慎

## 成交量模式

### 1. 稳定累积
- **模式**：OBV持续上升
- **含义**：持续买入压力
- **行动**：寻找买入机会
- **注意**：用价格确认验证

### 2. 稳定派发
- **模式**：OBV持续下降
- **含义**：持续卖出压力
- **行动**：避免买入，考虑做空
- **注意**：用价格确认验证

### 3. OBV激增
- **模式**：OBV急剧上升
- **含义**：重累积
- **行动**：潜在突破即将到来
- **注意**：可能是高潮，观察反转

### 4. OBV暴跌
- **模式**：OBV急剧下降
- **含义**：重派发
- **行动**：潜在跌破即将到来
- **注意**：可能是投降，观察底部

## 相关函数

- **AD**：累积/派发线 - 更复杂的成交量指标
- **ADOSC**：Chaikin A/D振荡器 - A/D线的衍生
- **MFI**：资金流量指数 - 将成交量与RSI概念结合
- **CMF**：Chaikin资金流量 - 短期成交量指标

## 附加说明

Joseph Granville在1963年的《Granville股票市场利润新钥匙》一书中介绍了平衡成交量。它是第一个系统地将成交量与价格变化联系起来的指标之一。

### 关键洞察

1. **Granville理论**：
   - "成交量先于价格"
   - 聪明钱在公众之前累积/派发
   - 成交量揭示聪明钱活动
   - OBV捕获这种流动

2. **早期警告系统**：
   - OBV变化通常发生在价格变化之前
   - 背离可以提前几天或几周信号
   - 不是时机工具而是警告指标
   - 仍需要价格行为进行入场时机

3. **背离可靠性**：
   - 在趋势市场中更可靠
   - 在区间市场中较不可靠
   - 与其他指标确认时更强
   - 最好伴随价格模式

4. **多时间框架使用**：
   - 更高时间框架用于整体成交量趋势
   - 更低时间框架用于入场时机
   - 两者应该对齐以获得最强信号
   - 更高TF的背离更显著

5. **板块分析**：
   - 比较板块股票的OBV
   - 识别领导力（最强OBV）
   - 识别落后者（最弱OBV）
   - 轮换到最强OBV股票

### 实用技巧

**对于多头头寸**：
- 希望看到上升OBV确认上升价格
- 在OBV保持强劲时在回调时入场
- 当OBV开始下跌时出场

**对于空头头寸**：
- 希望看到下降OBV确认下降价格
- 在OBV保持疲软时在反弹时入场
- 当OBV开始上升时出场

**对于突破**：
- 突破前上升OBV = 强信号
- 突破前下降OBV = 弱信号
- OBV与价格创新高 = 非常看涨
- OBV滞后价格 = 谨慎

OBV最有效时用作确认工具而不是独立信号生成器。其简单性既是其优势（易于理解）也是其弱点（比其他成交量指标更不细致）。
