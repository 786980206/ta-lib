# TYPPRICE - 典型价格

## 描述

典型价格是一个简单的价格指标，表示给定周期内最高价、最低价和收盘价的平均值。它比单独使用收盘价提供更全面的价格行为视图，因为它包含了该周期的完整价格范围。

## 类别
价格变换/数学运算符

## 计算

典型价格是最高价、最低价和收盘价的算术平均值：

### 公式
```
典型价格 = (最高价 + 最低价 + 收盘价) / 3
```

## 参数

无

## 输入
- **最高价**：`double[]`
- **最低价**：`double[]`
- **收盘价**：`double[]`

## 输出
- 典型价格值：`double[]`

## 解释

### 目的
- **代表性价格**：表示K线价格行为的单一值
- **成交量分析**：通常与成交量指标一起使用
- **公允价值**：近似该周期的"公平"或"平均"价格

### 用途
典型价格通常用作以下指标的输入：
1. **成交量指标**：资金流量指数（MFI）、成交量加权指标
2. **振荡器**：可以在计算中替代收盘价
3. **移动平均线**：平滑典型价格而不是收盘价
4. **累积/派发**：A/D计算的组成部分

## 使用示例

```c
// C/C++示例
double high[100], low[100], close[100];
double typprice[100];
int outBegIdx, outNBElement;

// 计算典型价格
TA_RetCode retCode = TA_TYPPRICE(
    0,                    // 开始索引
    99,                   // 结束索引
    high,                 // 最高价
    low,                  // 最低价
    close,                // 收盘价
    &outBegIdx,           // 输出：开始索引
    &outNBElement,        // 输出：元素数量
    typprice              // 输出：典型价格值
);
```

## 与其他价格变换的比较

| 价格类型 | 公式 | 用途 |
|----------|------|------|
| 收盘价 | 收盘价 | 最常见，周期结束 |
| 典型价格 | (H+L+C)/3 | 平衡，成交量指标 |
| 中位价格 | (H+L)/2 | 范围中点 |
| 加权收盘价 | (H+L+C+C)/4 | 强调收盘价 |
| 平均价格 | (O+H+L+C)/4 | 完整OHLC平均 |

## 优势

1. **全面**：包含完整价格范围
2. **简单**：易于计算和理解
3. **平衡**：对H、L、C等权重
4. **标准**：在指标中广泛使用

## 局限性

1. **等权重**：不强调收盘价
2. **无成交量**：不包含成交量数据
3. **简单平均**：可能不反映真正的"典型"价格

## 相关函数

- **MEDPRICE**：中位价格 (H+L)/2
- **WCLPRICE**：加权收盘价 (H+L+C+C)/4
- **AVGPRICE**：平均价格 (O+H+L+C)/4
- **CCI**：商品通道指数 - 使用典型价格
- **MFI**：资金流量指数 - 使用典型价格

## 附加说明

典型价格主要是其他指标的构建块，而不是独立的交易工具。它的主要价值在于提供单一的代表性价格，捕捉每个周期的最高价、最低价和收盘价，使其对成交量加权计算和受益于考虑完整价格范围而不是仅收盘价的指标有用。
