# MOM - 动量

## 描述

动量是技术分析中最简单和最基本的指标之一。它通过比较当前价格与n个周期前的价格来测量指定周期内价格的变化率。正动量表示向上价格移动，而负动量表示向下移动。

## 分类
动量指标

## 计算方法

动量计算为当前价格与n个周期前价格的简单差异：

### 公式
```
MOM = 价格(今天) - 价格(今天 - n)
```

其中：
- n = 时间周期（回望周期）
- 价格 = 通常是收盘价

### 示例
如果今天收盘价为52美元，10天前收盘价为50美元：
```
MOM(10) = 52 - 50 = 2
```

## 参数

- **optInTimePeriod**（默认：10）：回望的周期数
  - 有效范围：1到100000
  - 常用值：10、14、20

## 输入
- 价格数据：`double[]`（通常是收盘价）

## 输出
- 动量值：`double[]`（无界，可以是正值或负值）

## 解释

### 动量值
- **MOM > 0**：价格高于n个周期前（向上动量）
- **MOM = 0**：价格与n个周期前相同（无动量）
- **MOM < 0**：价格低于n个周期前（向下动量）

### 幅度
- **增加MOM**：动量加强
- **减少MOM**：动量减弱
- **大正MOM**：强向上动量
- **大负MOM**：强向下动量

### 交易信号

1. **零线交叉**：
   - **买入**：MOM交叉零线上方
   - **卖出**：MOM交叉零线下方

2. **背离**：
   - **看涨**：价格更低低点，MOM更高低点
   - **看跌**：价格更高高点，MOM更低高点

3. **趋势强度**：
   - 加速MOM = 强趋势
   - 减速MOM = 趋势减弱

4. **极值**：
   - 极高MOM = 潜在衰竭
   - 极低MOM = 潜在反转

## 使用示例

```c
// C/C++ 示例
double closePrices[100];
double momOutput[100];
int outBegIdx, outNBElement;

// 计算10周期动量
TA_RetCode retCode = TA_MOM(
    0,                    // 开始索引
    99,                   // 结束索引
    closePrices,          // 输入价格数据
    10,                   // 时间周期
    &outBegIdx,           // 输出：开始索引
    &outNBElement,        // 输出：元素数量
    momOutput             // 输出：动量值
);
```

## 实现细节

- 非常简单计算：价格(t) - 价格(t-n)
- 回望周期：n个周期
- 无平滑或标准化
- 输出与价格相同单位

## 优势

1. **简单**：最容易理解的动量指标
2. **快速**：无复杂计算
3. **清晰**：直接解释
4. **通用**：适用于所有市场和时间框架

## 限制

1. **未标准化**：值取决于价格规模
2. **无界限**：无定义的超买/超卖水平
3. **滞后**：基于历史价格比较
4. **鞭打**：可能产生假信号

## 相关函数

- **ROC**：变化率 - 动量的百分比版本
- **ROCP**：变化率百分比 - 另一个百分比版本
- **RSI**：相对强弱指数 - 标准化动量
- **CMO**：Chande动量振荡器 - 类似概念

## 附加说明

动量是许多更复杂指标的基础。虽然简单，但它有效地捕获了价格变化率。为了比较不同证券或时间段的动量，考虑使用ROC（基于百分比）代替。
