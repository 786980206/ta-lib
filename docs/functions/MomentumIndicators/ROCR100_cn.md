# ROCR100 - 变化率比率100

## 描述

变化率比率100（ROCR100）是一个动量指标，测量当前价格与n个周期前价格的比率，然后乘以100。它类似于ROCR但表示为百分比，使得更容易解释和比较不同证券。ROCR100有助于识别动量变化并提供价格移动的标准化测量。

## 分类
动量指标

## 作者
传统技术分析

## 计算方法

ROCR100计算为当前价格与过去价格的比率，乘以100：

### 公式
```
ROCR100 = (当前价格 / n个周期前的价格) × 100
```

其中：
- 当前价格 = 当前收盘价
- n个周期前的价格 = n个周期前的价格
- 结果表示为百分比

### 示例
如果当前价格为110，10个周期前的价格为100：
```
ROCR100 = (110 / 100) × 100 = 110%
```

## 参数

- **optInTimePeriod**（默认：10）：计算周期
  - 有效范围：1到100000
  - 常用值：10、14、20

## 输入
- 价格数据：`double[]`（通常是收盘价）

## 输出
- ROCR100值：`double[]`（百分比）

## 解释

### ROCR100值
- **> 100**：价格升值（上升趋势动量）
- **< 100**：价格贬值（下降趋势动量）
- **= 100**：价格无变化（无动量）
- **高值**：强上升趋势动量
- **低值**：强下降趋势动量

### 交易信号

1. **100线交叉**：
   - **买入**：ROCR100交叉100上方
   - **卖出**：ROCR100交叉100下方
   - **最适合**：动量变化检测

2. **动量变化**：
   - **上升ROCR100**：动量增加
   - **下降ROCR100**：动量减少
   - **峰值ROCR100**：动量峰值（潜在反转）
   - **谷值ROCR100**：动量谷值（潜在反转）

3. **背离**：
   - **看涨**：价格更低低点，ROCR100更高低点
   - **看跌**：价格更高高点，ROCR100更低高点
   - **最适合**：趋势衰竭点

4. **趋势强度**：
   - **强上升趋势**：ROCR100持续高于100且上升
   - **强下降趋势**：ROCR100持续低于100且下降
   - **弱趋势**：ROCR100在100周围振荡

## 使用示例

## 交易策略

### 1. 100线策略
- **买入**：ROCR100交叉100上方
- **卖出**：ROCR100交叉100下方
- **过滤器**：仅当|ROCR100 - 100| > 5时交易
- **最适合**：动量变化检测

### 2. 动量策略
- **买入**：ROCR100上升且高于100
- **卖出**：ROCR100下降且低于100
- **出场**：ROCR100动量变化
- **最适合**：趋势跟踪

### 3. 背离策略
- **设置**：识别价格和ROCR100之间的背离
- **确认**：等待价格模式
- **入场**：确认时
- **最适合**：趋势衰竭点

### 4. ROCR100 + 趋势策略
- **设置**：使用ROCR100进行时机
- **入场**：趋势方向与ROCR100时机
- **出场**：ROCR100反转或趋势变化
- **最适合**：趋势市场

## ROCR100 vs. ROCR

| 方面 | ROCR100 | ROCR |
|------|---------|------|
| 单位 | 百分比 | 比率 |
| 比较 | 跨证券容易 | 跨证券容易 |
| 缩放 | 自动 | 自动 |
| 解释 | 百分比变化 | 比率变化 |
| 最适合 | 跨证券分析 | 跨证券分析 |

## 优势

1. **标准化**：百分比使比较容易
2. **通用**：适用于所有证券
3. **可缩放**：无需手动缩放
4. **清晰**：易于理解的百分比
5. **投资组合**：对投资组合分析有用

## 限制

1. **仍有滞后**：基于历史数据
2. **鞭打**：在波动市场中可能发生
3. **周期敏感性**：结果因周期而异
4. **无方向**：不指示未来方向
5. **上下文依赖**：需要解释

## 周期选择

### 短周期（5-10）
- **特征**：更具响应性
- **使用**：短期交易
- **权衡**：更多信号，更多噪音
- **最适合**：主动交易

### 标准周期（10-20）
- **特征**：平衡方法
- **使用**：一般动量分析
- **权衡**：良好平衡
- **最适合**：大多数应用

### 长周期（20-50）
- **特征**：更平滑，响应性较低
- **使用**：长期分析
- **权衡**：较少信号，更可靠
- **最适合**：头寸交易

## 附加说明

ROCR100是作为ROCR的改进而开发的，通过使用百分比而不是比率，使得更容易解释。

### 关键洞察

1. **百分比优势**：
   - 容易跨证券比较
   - 无需手动缩放
   - 通用解释
   - 投资组合分析友好

2. **动量测量**：
   - 测量百分比动量
   - > 100 = 上升趋势动量
   - < 100 = 下降趋势动量
   - 100 = 无动量

3. **最佳应用**：
   - 跨证券动量分析
   - 投资组合动量评估
   - 趋势变化检测
   - 动量强度评估

4. **信号解释**：
   - 100线交叉 = 动量变化
   - 上升/下降 = 动量变化
   - 峰值/谷值 = 潜在反转
   - 背离 = 趋势衰竭

5. **组合策略**：
   - 与趋势指标一起使用
   - 与成交量分析结合
   - 用于动量分析
   - 多时间框架分析

### 实用技巧

**对于动量分析**：
- 跟踪ROCR100随时间变化
- 识别动量状态
- 用于动量交易
- 监控动量变化

**对于趋势跟踪**：
- 使用ROCR100确定趋势方向
- 在动量确认时入场
- 在动量损失时出场
- 用成交量确认

**对于背离交易**：
- 识别价格vs. ROCR100背离
- 等待确认
- 使用支撑/阻力
- 在极值外设置止损

**对于投资组合分析**：
- 比较跨证券的ROCR100
- 按动量强度排序
- 按动量多样化
- 用于资产配置

ROCR100对于需要比较多个证券动量或想要ROC标准化版本的交易者特别有价值。它非常适合投资组合分析并提供清晰的动量信号。
