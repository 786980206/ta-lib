# ADXR - 平均方向性移动指数评级

## 描述

平均方向性移动指数评级（ADXR）是一个趋势强度指标，用于衡量当前ADX和指定周期前ADX的平均值。它由J. Welles Wilder开发，作为ADX的平滑版本，减少噪音并提供更稳定的趋势强度读数。ADXR有助于识别趋势强度并提供更可靠的方向性移动衡量。

## 分类
动量指标

## 作者
J. Welles Wilder

## 计算方法

ADXR计算为当前ADX和过去ADX的平均值：

### 公式
```
ADXR = (ADX + ADX[n个周期前]) / 2
```

其中：
- ADX = 当前平均方向性指数
- ADX[n个周期前] = n个周期前的ADX
- n = 平滑周期（默认：14）

### 示例
如果当前ADX = 25，14个周期前的ADX = 20：
```
ADXR = (25 + 20) / 2 = 22.5
```

## 参数

- **optInTimePeriod**（默认：14）：ADX计算的周期
  - 有效范围：2 到 100000
  - 常用值：14（标准）、21、30

## 输入
- **最高价**：`double[]`
- **最低价**：`double[]`
- **收盘价**：`double[]`

## 输出
- ADXR 数值：`double[]`（范围：0 到 100）

## 解释

### ADXR 数值
- **0-25**：弱趋势
- **25-50**：中等趋势
- **50-75**：强趋势
- **75-100**：非常强趋势

### 交易信号

1. **趋势强度**：
   - **弱趋势**：ADXR < 25
   - **中等趋势**：ADXR 25-50
   - **强趋势**：ADXR 50-75
   - **非常强趋势**：ADXR > 75

2. **趋势变化**：
   - **上升ADXR**：趋势加强
   - **下降ADXR**：趋势减弱
   - **ADXR峰值**：趋势峰值（潜在反转）
   - **ADXR谷值**：趋势谷值（潜在反转）

3. **背离**：
   - **看涨**：价格更低低点，ADXR更高低点
   - **看跌**：价格更高高点，ADXR更低高点
   - **最适合**：趋势衰竭点

4. **趋势确认**：
   - **高ADXR**：强趋势确认
   - **低ADXR**：弱趋势或横盘
   - **上升ADXR**：趋势加强
   - **下降ADXR**：趋势减弱

## 使用示例

```c
// C/C++ 示例
double high[100], low[100], close[100];
double adxrOutput[100];
int outBegIdx, outNBElement;

// 计算14周期ADXR
TA_RetCode retCode = TA_ADXR(
    0,                    // 开始索引
    99,                   // 结束索引
    high,                 // 最高价
    low,                  // 最低价
    close,                // 收盘价
    14,                   // 时间周期
    &outBegIdx,           // 输出：开始索引
    &outNBElement,        // 输出：元素数量
    adxrOutput            // 输出：ADXR数值
);
```

## 实现细节

TA-Lib ADXR 实现：

1. **ADX计算**：计算当前ADX
2. **过去ADX**：获取n个周期前的ADX
3. **平均**：平均当前和过去的ADX
4. **回望**：需要2×n个周期才能产生第一个输出

## 交易策略

### 1. 趋势强度策略
- **设置**：跟踪ADXR随时间变化
- **入场**：当ADXR显示强趋势时
- **出场**：当ADXR显示弱趋势时
- **最适合**：趋势跟踪

### 2. 趋势变化策略
- **设置**：识别ADXR变化
- **入场**：趋势加强时
- **出场**：趋势减弱时
- **最适合**：趋势变化检测

### 3. 背离策略
- **设置**：识别价格和ADXR之间的背离
- **确认**：等待价格模式
- **入场**：确认时
- **最适合**：趋势衰竭点

### 4. ADXR + 方向策略
- **设置**：使用ADXR进行趋势强度
- **入场**：方向与ADXR确认
- **出场**：ADXR反转或方向变化
- **最适合**：趋势跟踪

## ADXR vs. ADX

| 方面 | ADXR | ADX |
|------|------|-----|
| 平滑 | 双重平滑 | 单一平滑 |
| 稳定性 | 更稳定 | 不太稳定 |
| 信号 | 更少，更可靠 | 更多，不太可靠 |
| 噪音 | 更少噪音 | 更多噪音 |
| 最适合 | 趋势跟踪 | 短期分析 |

## 优势

1. **稳定**：双重平滑减少噪音
2. **可靠**：比ADX更少假信号
3. **通用**：适用于所有市场
4. **清晰**：易于解释
5. **多功能**：多种应用

## 限制

1. **不太敏感**：比ADX更慢
2. **仍然滞后**：基于历史数据
3. **震荡**：在震荡市场中可能发生
4. **周期敏感性**：结果随周期变化
5. **学习曲线**：需要理解概念

## 周期选择

### 短期（10-14）
- **特征**：更敏感
- **用途**：短期交易
- **权衡**：更多信号，更多噪音
- **最适合**：主动交易

### 标准期（14-21）
- **特征**：平衡方法
- **用途**：一般趋势分析
- **权衡**：良好平衡
- **最适合**：大多数应用

### 长期（21-30）
- **特征**：更平滑，不太敏感
- **用途**：长期分析
- **权衡**：更少信号，更可靠
- **最适合**：头寸交易

## 附加说明

J. Welles Wilder开发了ADXR作为ADX的平滑版本，以减少噪音并提供更稳定的趋势强度读数。

### 关键洞察

1. **双重平滑**：
   - 平均当前和过去的ADX
   - 减少噪音同时保持趋势信息
   - 比ADX更稳定
   - 更适合趋势跟踪

2. **趋势强度测量**：
   - 测量平均趋势强度
   - 更高ADXR = 更强趋势
   - 更低ADXR = 更弱趋势
   - 上升ADXR = 趋势加强

3. **最佳应用**：
   - 趋势强度分析
   - 趋势跟踪
   - 趋势变化检测
   - 趋势确认

4. **信号解释**：
   - < 25 = 弱趋势
   - 25-50 = 中等趋势
   - 50-75 = 强趋势
   - > 75 = 非常强趋势

5. **组合策略**：
   - 与方向指标一起使用
   - 结合价格分析
   - 用于趋势确认
   - 多时间框架分析

### 实用技巧

**趋势强度分析**：
- 跟踪ADXR随时间变化
- 识别趋势强度状态
- 用于趋势确认
- 监控趋势变化

**趋势跟踪**：
- 使用ADXR进行趋势强度
- 在强趋势确认时入场
- 在趋势减弱时出场
- 使用方向指标确定方向

**趋势变化检测**：
- 观察ADXR变化
- 上升ADXR = 趋势加强
- 下降ADXR = 趋势减弱
- 用于趋势反转信号

**风险管理**：
- 使用ADXR进行头寸规模
- 在弱趋势期间减少规模
- 在强趋势期间增加规模
- 用于止损放置

ADXR对于想要比ADX更稳定的趋势强度指标的交易者特别有价值。它非常适合趋势跟踪，并为趋势强度变化提供更可靠的信号。
