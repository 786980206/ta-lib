# RSI - 相对强弱指数

## 描述

相对强弱指数（RSI）是一个动量振荡器，测量价格变化的速度和幅度。它在0和100之间振荡，主要用于识别市场中的超买或超卖条件。

## 分类
动量指标

## 作者
J. Welles Wilder

## 计算方法

RSI使用以下步骤计算：

1. **计算价格变化**：对于每个周期，计算连续收盘价之间的差异
2. **分离收益和损失**：
   - 收益 = 如果价格变化为正则为价格变化，否则为0
   - 损失 = 如果价格变化为负则为价格变化的绝对值，否则为0

3. **计算平均收益和平均损失**：
   - 初始平均收益 = 周期内收益总和 / 周期
   - 初始平均损失 = 周期内损失总和 / 周期

4. **平滑平均值**（Wilder平滑方法）：
   - 平均收益 = [(前一个平均收益) × (周期-1) + 当前收益] / 周期
   - 平均损失 = [(前一个平均损失) × (周期-1) + 当前损失] / 周期

5. **计算RS（相对强度）**：
   - RS = 平均收益 / 平均损失

6. **计算RSI**：
   - RSI = 100 - (100 / (1 + RS))
   - 或等效地：RSI = 100 × (平均收益 / (平均收益 + 平均损失))

### 公式
```
RSI = 100 × (平均收益 / (平均收益 + 平均损失))
```

## 参数

- **optInTimePeriod**（默认：14）：用于计算RSI的周期数
  - 有效范围：2到100000
  - 常用值：9、14、25

## 输入
- 价格数据（通常是收盘价）：`double[]`

## 输出
- RSI值：`double[]`（范围：0到100）

## 解释

### 传统水平
- **超买**：RSI > 70（价格可能过高，需要回调）
- **超卖**：RSI < 30（价格可能过低，需要反弹）
- **中性**：RSI约50

### 交易信号
1. **超买/超卖**：
   - 当RSI交叉70上方时考虑卖出
   - 当RSI交叉30下方时考虑买入

2. **背离**：
   - **看涨背离**：价格创更低低点而RSI创更高低点（潜在向上反转）
   - **看跌背离**：价格创更高高点而RSI创更低高点（潜在向下反转）

3. **中心线交叉**：
   - RSI交叉50上方可能表示看涨动量
   - RSI交叉50下方可能表示看跌动量

4. **失败摆动**：
   - 看涨：RSI跌破30，反弹，回调但保持在30上方，然后突破其前期高点
   - 看跌：RSI升至70上方，下跌，反弹但保持在70下方，然后跌破其前期低点

### 周期选择
- **较短周期**（例如9）：更敏感，产生更多信号但更多假信号
- **较长周期**（例如25）：不太敏感，较少信号但更可靠
- **标准周期**（14）：平衡方法，最广泛使用


## 实现细节

TA-Lib实现遵循Wilder的原始算法，具有以下特征：

1. **平滑方法**：使用Wilder的指数平滑（不同于标准EMA）
2. **回望周期**：在产生第一个输出之前需要`timePeriod`个柱
3. **不稳定周期**：可通过`TA_SetUnstablePeriod`为RSI配置
4. **MetaStock兼容性**：启用兼容模式时，计算提前一个柱开始
5. **零除法处理**：当平均收益和平均损失都为零时返回0

## 限制和考虑

1. **滞后指标**：RSI基于历史价格，滞后当前价格行为
2. **趋势中的假信号**：在强趋势期间，RSI可能在超买或超卖区域保持较长时间
3. **时间框架敏感性**：结果因所选周期而异
4. **非独立指标**：应与其他技术分析工具结合使用
5. **鞭打**：在波动、横盘市场中可能产生假信号

## 附加说明

RSI是最受欢迎的技术指标之一，几乎包含在所有图表和技术分析软件中。J. Welles Wilder在1978年的书中介绍了它，并建议使用14天周期。该指标特别适用于：

- 识别潜在反转点
- 确认趋势强度
- 发现可能先于趋势变化的背离
- 在区间市场中时机入场和出场点

使用RSI时，交易者经常根据市场条件调整超买/超卖水平。在强上升趋势中，超买水平可能提高到80，而在强下降趋势中，超卖水平可能降低到20。
