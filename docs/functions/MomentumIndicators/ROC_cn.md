# ROC - 变化率

## 描述

变化率（ROC）是一个动量指标，测量当前价格与n个周期前价格之间的百分比变化。与显示绝对变化的动量指标不同，ROC显示相对（百分比）变化，使得更容易比较不同证券和时间段。

## 分类
动量指标

## 计算方法

ROC计算从n个周期前的百分比变化：

### 公式
```
ROC = ((价格(今天) / 价格(今天 - n)) - 1) × 100
```

或等效地：
```
ROC = ((价格(今天) - 价格(今天 - n)) / 价格(今天 - n)) × 100
```

### 示例
如果今天收盘价为55美元，10天前收盘价为50美元：
```
ROC(10) = ((55 / 50) - 1) × 100 = 0.10 × 100 = 10%
```

## 参数

- **optInTimePeriod**（默认：10）：回望的周期数
  - 有效范围：1到100000
  - 常用值：10、12、20、25

## 输入
- 价格数据：`double[]`（通常是收盘价）

## 输出
- ROC值：`double[]`（百分比，无界）

## 解释

### ROC值
- **ROC > 0**：价格高于n个周期前（正动量）
- **ROC = 0**：价格与n个周期前相同
- **ROC < 0**：价格低于n个周期前（负动量）

### 常见阈值（因证券而异）
- **ROC > +5%**：强正动量
- **ROC < -5%**：强负动量
- 可根据证券波动性调整

### 交易信号

1. **零线交叉**：
   - **买入**：ROC交叉零线上方
   - **卖出**：ROC交叉零线下方

2. **超买/超卖**（阈值因证券而异）：
   - **超买**：ROC > +10%（或更高）
   - **超卖**：ROC < -10%（或更低）

3. **背离**：
   - **看涨**：价格更低低点，ROC更高低点
   - **看跌**：价格更高高点，ROC更低高点

4. **趋势强度**：
   - 增加ROC = 加速趋势
   - 减少ROC = 减速趋势

## 使用示例

```c
// C/C++ 示例
double closePrices[100];
double rocOutput[100];
int outBegIdx, outNBElement;

// 计算10周期ROC
TA_RetCode retCode = TA_ROC(
    0,                    // 开始索引
    99,                   // 结束索引
    closePrices,          // 输入价格数据
    10,                   // 时间周期
    &outBegIdx,           // 输出：开始索引
    &outNBElement,        // 输出：元素数量
    rocOutput             // 输出：ROC值
);
```

## 实现细节

- 计算百分比变化
- 输出为百分比（例如，10.5表示10.5%）
- 回望周期：n个周期
- 可用于比较不同证券

## 优势

1. **标准化**：百分比使比较容易
2. **通用**：适用于所有证券
3. **清晰**：易于理解（百分比变化）
4. **多功能**：多种交易应用

## 限制

1. **鞭打**：可能产生假信号
2. **滞后**：基于历史比较
3. **无标准水平**：超买/超卖因证券而异
4. **零除法**：如果过去价格为0则未定义

## 相关函数

- **MOM**：动量 - 绝对变化版本
- **ROCP**：变化率百分比 - 略有不同的公式
- **ROCR**：变化率比率 - 比率版本
- **ROCR100**：变化率比率100刻度
- **RSI**：相对强弱指数 - 标准化替代

## 附加说明

当比较多个证券或价格规模在不同时间段差异很大时，ROC优于动量。百分比基础提供了标准化的动量视图，更容易解释和比较。
