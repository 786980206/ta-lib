# DX - 方向性移动

## 描述

方向性移动（DX）是一个趋势强度指标，用于测量方向性移动的强度而不进行平滑。它是用于推导ADX的原始计算，提供更敏感的趋势强度测量。DX有助于识别趋势强度并提供趋势变化的早期信号。

## 分类
动量指标

## 作者
J. Welles Wilder

## 计算方法

DX使用方向性移动组成部分计算：

### 步骤1：计算正方向性移动（+DM）
```
+DM = 最高价 - 前最高价（如果为正，否则为0）
```

### 步骤2：计算负方向性移动（-DM）
```
-DM = 前最低价 - 最低价（如果为正，否则为0）
```

### 步骤3：计算真实区间（TR）
```
TR = max(最高价 - 最低价, |最高价 - 前收盘价|, |最低价 - 前收盘价|)
```

### 步骤4：计算平滑组成部分
```
平滑+DM = EMA(+DM, 周期)
平滑-DM = EMA(-DM, 周期)
平滑TR = EMA(TR, 周期)
```

### 步骤5：计算DX
```
DX = |平滑+DM - 平滑-DM| / (平滑+DM + 平滑-DM) × 100
```

## 参数

- **optInTimePeriod**（默认：14）：计算周期
  - 有效范围：2 到 100000
  - 常用值：14（标准）、21、30

## 输入
- **最高价**：`double[]`
- **最低价**：`double[]`
- **收盘价**：`double[]`

## 输出
- DX 数值：`double[]`（范围：0到100）

## 解释

### DX 数值
- **0-25**：弱方向性移动
- **25-50**：中等方向性移动
- **50-75**：强方向性移动
- **75-100**：非常强方向性移动

### 交易信号

1. **趋势强度**：
   - **弱趋势**：DX < 25
   - **中等趋势**：DX 25-50
   - **强趋势**：DX 50-75
   - **非常强趋势**：DX > 75

2. **趋势变化**：
   - **上升DX**：趋势加强
   - **下降DX**：趋势减弱
   - **DX峰值**：趋势峰值（潜在反转）
   - **DX谷值**：趋势谷值（潜在反转）

3. **背离**：
   - **看涨**：价格更低低点，DX更高低点
   - **看跌**：价格更高高点，DX更低高点
   - **最适合**：趋势衰竭点

4. **趋势确认**：
   - **高DX**：强趋势确认
   - **低DX**：弱趋势或横盘
   - **上升DX**：趋势加强
   - **下降DX**：趋势减弱

## 使用示例

```c
// C/C++ 示例
double high[100], low[100], close[100];
double dxOutput[100];
int outBegIdx, outNBElement;

// 计算14周期DX
TA_RetCode retCode = TA_DX(

    99,                   // 结束索引

    low,                  // 最低价
    close,                // 收盘价
    14,                   // 时间周期
    &outBegIdx,           // 输出：开始索引
    &outNBElement,        // 输出：元素数量
    dxOutput              // 输出：DX数值
);
```

## 实现细节

TA-Lib DX 实现：

1. **方向性移动**：计算每个周期的+DM和-DM
2. **真实区间**：计算每个周期的TR
3. **平滑**：对所有组成部分应用EMA
4. **DX计算**：计算比率并转换为百分比
5. **回望**：需要n个周期才能产生第一个输出

## 交易策略

### 1. 趋势强度策略
- **设置**：跟踪DX随时间变化
- **入场**：当DX显示强趋势时
- **出场**：当DX显示弱趋势时
- **最适合**：趋势跟踪

### 2. 趋势变化策略
- **设置**：识别DX变化
- **入场**：趋势加强时
- **出场**：趋势减弱时
- **最适合**：趋势变化检测

### 3. 背离策略
- **设置**：识别价格和DX之间的背离
- **确认**：等待价格模式
- **入场**：确认时
- **最适合**：趋势衰竭点

### 4. DX + 方向策略
- **设置**：使用DX进行趋势强度
- **入场**：方向与DX确认
- **出场**：DX反转或方向变化
- **最适合**：趋势跟踪

## DX vs. ADX

| 方面 | DX | ADX |
|------|----|----|
| 平滑 | 单一平滑 | 双重平滑 |
| 响应性 | 更敏感 | 不太敏感 |
| 信号 | 更多信号 | 更少信号 |
| 噪音 | 更多噪音 | 更少噪音 |
| 最适合 | 短期分析 | 长期分析 |

## 优势

1. **敏感**：比ADX更敏感
2. **早期信号**：提供早期趋势变化
3. **通用**：适用于所有市场
4. **清晰**：易于解释
5. **多功能**：多种应用

## 限制

1. **噪音**：比ADX更波动
2. **震荡**：可能产生假信号
3. **周期敏感性**：结果随周期变化
4. **上下文依赖**：需要解释
5. **学习曲线**：需要理解概念

## 周期选择

### 短期（10-14）
- **特征**：更敏感
- **用途**：短期交易
- **权衡**：更多信号，更多噪音
- **最适合**：主动交易

### 标准期（14-21）
- **特征**：平衡方法
- **用途**：一般趋势分析
- **权衡**：良好平衡
- **最适合**：大多数应用

### 长期（21-30）
- **特征**：更平滑，不太敏感
- **用途**：长期分析
- **权衡**：更少信号，更可靠
- **最适合**：头寸交易

## 附加说明

J. Welles Wilder开发了DX作为方向性移动强度的原始计算，然后平滑以创建ADX。

### 关键洞察

1. **原始计算**：
   - 测量方向性移动强度
   - 更高DX = 更强趋势
   - 更低DX = 更弱趋势
   - 上升DX = 趋势加强

2. **趋势强度测量**：
   - 测量绝对趋势强度
   - 无方向指示
   - 与PLUS_DI/MINUS_DI一起使用确定方向
   - 对趋势确认有用

3. **最佳应用**：
   - 趋势强度分析
   - 趋势变化检测
   - 趋势确认
   - 早期趋势信号

4. **信号解释**：
   - < 25 = 弱趋势
   - 25-50 = 中等趋势
   - 50-75 = 强趋势
   - > 75 = 非常强趋势

5. **组合策略**：
   - 与方向指标一起使用
   - 结合价格分析
   - 用于趋势确认
   - 多时间框架分析

### 实用技巧

**趋势强度分析**：
- 跟踪DX随时间变化
- 识别趋势强度状态
- 用于趋势确认
- 监控趋势变化

**趋势变化检测**：
- 观察DX变化
- 上升DX = 趋势加强
- 下降DX = 趋势减弱
- 用于早期信号

**趋势确认**：
- 使用DX进行趋势强度
- 结合方向指标
- 在强趋势确认时入场
- 在趋势减弱时出场

**风险管理**：
- 使用DX进行头寸规模
- 在弱趋势期间减少规模
- 在强趋势期间增加规模
- 用于止损放置

DX对于想要比ADX更敏感的趋势强度指标的交易者特别有价值。它非常适合短期趋势分析，并为趋势变化提供早期信号。
