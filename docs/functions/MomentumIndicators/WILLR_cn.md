# WILLR - 威廉%R

## 描述

威廉%R，也称为威廉百分比范围，是一个测量超买和超卖水平的动量指标。它类似于随机振荡器，但在0到-100的反向刻度上绘制。由拉里·威廉姆斯开发，它显示当前收盘价与回望期间最高价之间的关系。

## 类别
动量指标

## 作者
拉里·威廉姆斯

## 计算

威廉%R将当前收盘价与周期内的高低范围进行比较：

### 公式
```
%R = (最高价 - 收盘价) / (最高价 - 最低价) × -100
```

其中：
- 最高价 = n周期内的最高价
- 最低价 = n周期内的最低价
- 收盘价 = 当前收盘价

结果范围从0到-100（反向刻度）。

## 参数

- **optInTimePeriod**（默认：14）：回望周期
  - 有效范围：2到100000
  - 常用值：14（标准）、10、20

## 输入
- **最高价**：`double[]`
- **最低价**：`double[]`
- **收盘价**：`double[]`

## 输出
- 威廉%R值：`double[]`（范围：0到-100）

## 解释

### 水平（反向刻度）
- **-20到0**：超买（价格接近范围顶部）
- **-50**：中点
- **-80到-100**：超卖（价格接近范围底部）

### 交易信号

1. **超买/超卖**：
   - **超买**：%R > -20（价格接近高点）
   - **超卖**：%R < -80（价格接近低点）
   - **卖出**：当%R退出超买区域（向下穿越-20）
   - **买入**：当%R退出超卖区域（向上穿越-80）

2. **失败摆动**：
   - **看涨**：%R未能达到超卖，然后反弹
   - **看跌**：%R未能达到超买，然后下跌

3. **背离**：
   - **看涨**：价格更低低点，%R更高低点（不太超卖）
   - **看跌**：价格更高高点，%R更低高点（不太超买）

4. **动量变化**：
   - %R的急剧变动表示强动量
   - %R的平坦化表示动量减弱

## 使用示例

```c
// C/C++示例
double high[100], low[100], close[100];
double willrOutput[100];
int outBegIdx, outNBElement;

// 计算14周期威廉%R
TA_RetCode retCode = TA_WILLR(
    0,                    // 开始索引
    99,                   // 结束索引
    high,                 // 最高价
    low,                  // 最低价
    close,                // 收盘价
    14,                   // 时间周期
    &outBegIdx,           // 输出：开始索引
    &outNBElement,        // 输出：元素数量
    willrOutput           // 输出：威廉%R值
);
```

## 实现细节

TA-Lib WILLR实现：

1. **范围计算**：找到周期内的最高价和最低价
2. **位置计算**：确定收盘价在范围内的位置
3. **反向**：乘以-100（反向刻度）
4. **回望**：需要n个周期的数据

## 与随机指标的关系

威廉%R与快速随机%K密切相关：

```
威廉%R = -100 - 随机%K（未平滑）

或者：
随机%K = 威廉%R + 100
```

主要区别：
- **刻度**：威廉%R：0到-100，随机：0到100
- **平滑**：随机通常平滑，威廉%R原始
- **水平**：反向阈值

## 交易策略

### 1. 超买/超卖策略
- **买入**：%R从下方向上穿越-80
- **卖出**：%R从上方向下穿越-20
- **止损**：最近摆动高点/低点
- **目标**：相反极值
- **最佳用于**：区间市场

### 2. 威廉%R趋势跟踪
- **上升趋势**：仅采用买入信号（%R从超卖）
- **下降趋势**：仅采用卖出信号（%R从超买）
- **过滤**：使用MA或趋势线确定趋势
- **最佳用于**：趋势市场

### 3. 背离策略
- **设置**：识别价格和%R之间的背离
- **确认**：等待支撑/阻力突破
- **入场**：确认时
- **止损**：最近摆动之外
- **最佳用于**：趋势衰竭点

### 4. 多时间框架
- **更高时间框架**：确定整体趋势
- **更低时间框架**：使用%R进行入场时机选择
- **规则**：仅在时间框架一致时交易
- **最佳用于**：所有市场条件

## 优势

1. **清晰水平**：定义超买/超卖区域
2. **快速**：对价格变化反应迅速
3. **简单**：易于理解和使用
4. **有界**：始终在0和-100之间
5. **多功能**：适用于所有时间框架

## 局限性

1. **趋势中的虚假信号**：在强趋势中可能保持超买/超卖
2. **震荡**：在震荡市场中多次交叉
3. **无方向信息**：仅显示在范围内的相对位置
4. **滞后成分**：基于历史高低点
5. **周期敏感**：结果随周期选择变化

## 周期选择

### 不同周期
- **短（5-10）**：更敏感，更多信号
  - 日内交易
  - 快速反转
  - 更多虚假信号

- **标准（14）**：平衡方法
  - 拉里·威廉姆斯的原始设置
  - 最常见
  - 一般交易

- **长（20-28）**：更平滑，更少信号
  - 摆动交易
  - 头寸交易
  - 更可靠的信号

## 相关函数

- **STOCH**：随机振荡器 - 类似概念，不同刻度
- **STOCHF**：快速随机 - 与%R几乎相同
- **RSI**：相对强弱指数 - 另一个超买/超卖指标
- **CCI**：商品通道指数 - 类似应用

## 附加说明

拉里·威廉姆斯在1960年代开发了威廉%R。由于其简单性和有效性，它迅速流行起来。威廉姆斯本人是一位传奇交易者，他在12个月内以+11,376%的回报率赢得了世界期货交易锦标赛。

### 关键洞察

1. **反向刻度心理学**：
   - 由于历史原因，值为负数
   - 一些交易者觉得这令人困惑
   - 将-20视为"高"，-80视为"低"
   - 或者，加100转换为0-100刻度

2. **快速vs慢速随机**：
   - 威廉%R = 快速随机未平滑
   - 快速信号但更多噪音
   - 没有像随机那样的信号线
   - 更适合短期交易

3. **强趋势**：
   - 在上升趋势中，%R可以保持在-20以上（超买）
   - 在下降趋势中，%R可以保持在-80以下（超卖）
   - 不要逆强趋势
   - 用于入场时机选择，不是反转交易

4. **确认是关键**：
   - 不要单独交易%R信号
   - 用以下方式确认：
     * 价格模式
     * 支撑/阻力
     * 趋势指标
     * 成交量

5. **多重威廉%R**：
   - 同时使用多个周期
   - 例如，14周期和28周期
   - 当两者一致时信号最强
   - 有助于过滤虚假信号

### 实用技巧

**多头入场**：
- 等待%R < -80（超卖）
- 当%R向上穿越-80时入场
- 或者等待%R达到-50（动量确认）
- 在最近低点下方设置止损

**空头入场**：
- 等待%R > -20（超买）
- 当%R向下穿越-20时入场
- 或者等待%R跌至-50（动量确认）
- 在最近高点上方设置止损

**避免震荡**：
- 与趋势指标结合使用
- 仅在下降趋势中交易超买
- 仅在上升趋势中交易超卖
- 要求成交量确认

威廉%R特别适用于在既定趋势中时机选择入场和在区间市场中识别短期超买/超卖条件。
