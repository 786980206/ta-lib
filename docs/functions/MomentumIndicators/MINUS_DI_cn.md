# MINUS_DI - 负方向指标

## 描述

负方向指标（MINUS_DI）是一个趋势指标，测量向下价格移动的强度。它是J. Welles Wilder开发的方向移动系统的一部分，与PLUS_DI一起工作以识别趋势方向和强度。MINUS_DI有助于识别看跌趋势并提供向下价格移动的信号。

## 分类
动量指标

## 作者
J. Welles Wilder

## 计算方法

MINUS_DI使用方向移动进行计算：

### 步骤1：计算负方向移动（-DM）
```
-DM = 前一个低点 - 低点（如果为正，否则为0）
```

### 步骤2：计算真实范围（TR）
```
TR = max(最高价 - 最低价, |最高价 - 前一个收盘价|, |最低价 - 前一个收盘价|)
```

### 步骤3：计算平滑-DM
```
平滑-DM = EMA(-DM, 周期)
```

### 步骤4：计算平滑TR
```
平滑TR = EMA(TR, 周期)
```

### 步骤5：计算MINUS_DI
```
MINUS_DI = (平滑-DM / 平滑TR) × 100
```

## 参数

- **optInTimePeriod**（默认：14）：计算周期
  - 有效范围：1到100000
  - 常用值：14（标准）、21、30

## 输入
- **最高价**：`double[]`
- **最低价**：`double[]`
- **收盘价**：`double[]`

## 输出
- MINUS_DI值：`double[]`（范围：0到100）

## 解释

### MINUS_DI值
- **0-25**：弱向下移动
- **25-50**：中等向下移动
- **50-75**：强向下移动
- **75-100**：非常强向下移动

### 交易信号

1. **趋势方向**：
   - **MINUS_DI > PLUS_DI**：下降趋势
   - **MINUS_DI < PLUS_DI**：上升趋势
   - **MINUS_DI = PLUS_DI**：横盘

2. **趋势强度**：
   - **高MINUS_DI**：强向下移动
   - **低MINUS_DI**：弱向下移动
   - **上升MINUS_DI**：向下移动加强
   - **下降MINUS_DI**：向下移动减弱

3. **交叉**：
   - **买入**：MINUS_DI交叉PLUS_DI下方
   - **卖出**：MINUS_DI交叉PLUS_DI上方
   - **最适合**：趋势变化检测

4. **背离**：
   - **看涨**：价格更低低点，MINUS_DI更高低点
   - **看跌**：价格更高高点，MINUS_DI更低高点
   - **最适合**：趋势衰竭点

## 使用示例

```c
// C/C++ 示例
```

## 实现细节

TA-Lib MINUS_DI实现：

1. **方向移动**：计算每个周期的-DM
2. **真实范围**：计算每个周期的TR
3. **平滑**：对-DM和TR应用EMA
4. **MINUS_DI**：计算比率并转换为百分比
5. **回望**：第一个输出需要n个周期

## 交易策略

### 1. 趋势方向策略
- **买入**：MINUS_DI < PLUS_DI
- **卖出**：MINUS_DI > PLUS_DI
- **持有**：当MINUS_DI = PLUS_DI时
- **最适合**：趋势跟踪

### 2. 交叉策略
- **买入**：MINUS_DI交叉PLUS_DI下方
- **卖出**：MINUS_DI交叉PLUS_DI上方
- **过滤器**：仅当|MINUS_DI - PLUS_DI| > 5时交易
- **最适合**：趋势变化检测

### 3. 背离策略
- **设置**：识别价格和MINUS_DI之间的背离
- **确认**：等待价格模式
- **入场**：确认时
- **最适合**：趋势衰竭点

### 4. MINUS_DI + ADX策略
- **设置**：使用MINUS_DI进行方向，ADX进行强度
- **入场**：方向与强度确认
- **出场**：方向变化或强度损失
- **最适合**：趋势跟踪

## MINUS_DI vs. PLUS_DI

| 方面 | MINUS_DI | PLUS_DI |
|------|----------|---------|
| 方向 | 向下移动 | 向上移动 |
| 信号 | 看跌 | 看涨 |
| 交叉 | 上方 = 下降趋势 | 上方 = 上升趋势 |
| 最适合 | 下降趋势识别 | 上升趋势识别 |

## 优势

1. **方向性**：清晰的趋势方向信号
2. **通用**：适用于所有市场
3. **清晰**：易于解释
4. **通用**：许多应用
5. **可靠**：较少假信号

## 限制

1. **仍有滞后**：基于历史数据
2. **鞭打**：在波动市场中可能发生
3. **周期敏感性**：结果因周期而异
4. **无幅度**：不测量价格移动大小
5. **上下文依赖**：需要解释

## 周期选择

### 短周期（10-14）
- **特征**：更具响应性
- **使用**：短期交易
- **权衡**：更多信号，更多噪音
- **最适合**：主动交易

### 标准周期（14-21）
- **特征**：平衡方法
- **使用**：一般趋势分析
- **权衡**：良好平衡
- **最适合**：大多数应用

### 长周期（21-30）
- **特征**：更平滑，响应性较低
- **使用**：长期分析
- **权衡**：较少信号，更可靠
- **最适合**：头寸交易

## 附加说明

J. Welles Wilder开发MINUS_DI作为方向移动系统的一部分，用于测量向下价格移动强度。

### 关键洞察

1. **方向移动**：
   - 测量向下价格移动
   - 更高MINUS_DI = 更强下降趋势
   - 更低MINUS_DI = 更弱下降趋势
   - 上升MINUS_DI = 下降趋势加强

2. **趋势识别**：
   - MINUS_DI > PLUS_DI = 下降趋势
   - MINUS_DI < PLUS_DI = 上升趋势
   - MINUS_DI = PLUS_DI = 横盘
   - 用于趋势方向

3. **最佳应用**：
   - 趋势方向识别
   - 趋势强度分析
   - 趋势变化检测
   - 趋势确认

4. **信号解释**：
   - < 25 = 弱向下移动
   - 25-50 = 中等向下移动
   - 50-75 = 强向下移动
   - > 75 = 非常强向下移动

5. **组合策略**：
   - 与PLUS_DI一起使用进行方向
   - 与ADX结合进行强度
   - 用于趋势确认
   - 多时间框架分析

### 实用技巧

**对于趋势方向**：
- 使用MINUS_DI与PLUS_DI进行方向
- MINUS_DI > PLUS_DI = 下降趋势
- MINUS_DI < PLUS_DI = 上升趋势
- 观察交叉

**对于趋势强度**：
- 跟踪MINUS_DI随时间变化
- 上升MINUS_DI = 加强
- 下降MINUS_DI = 减弱
- 用于趋势确认

**对于趋势变化**：
- 观察MINUS_DI交叉
- 用价格行为确认
- 用成交量验证
- 在最近极值外设置止损

**对于风险管理**：
- 使用MINUS_DI进行头寸规模
- 在弱趋势期间减少规模
- 在强趋势期间增加规模
- 用于止损放置

MINUS_DI对于想要识别趋势方向和强度的交易者特别有价值。它非常适合趋势跟踪，并为向下价格移动提供清晰的信号。
