# ADX - 平均方向性移动指数

## 描述

平均方向性移动指数（ADX）是由 J. Welles Wilder 开发的趋势强度指标。ADX 量化趋势强度而不考虑方向，在 0 到 100 之间振荡。较高的值表示更强的趋势（上升或下降），而较低的值表示较弱的趋势或横盘市场。

## 分类
动量指标 / 趋势强度

## 作者
J. Welles Wilder

## 计算方法

ADX 来源于另外两个指标：正方向性指标（+DI）和负方向性指标（-DI）。

### 步骤1：计算方向性移动
```
+DM = 最高价(今天) - 最高价(昨天)  (如果为正且 > -DM，否则为0)
-DM = 最低价(昨天) - 最低价(今天)    (如果为正且 > +DM，否则为0)
```

### 步骤2：计算真实区间（TR）
```
TR = max[(最高价 - 最低价), |最高价 - 收盘价(前)|, |最低价 - 收盘价(前)|]
```

### 步骤3：平滑DM和TR（使用Wilder平滑）
```
+DM(平滑) = +DM(平滑前) - (+DM(平滑前)/n) + +DM(当前)
-DM(平滑) = -DM(平滑前) - (-DM(平滑前)/n) + -DM(当前)
TR(平滑) = TR(平滑前) - (TR(平滑前)/n) + TR(当前)
```

### 步骤4：计算方向性指标
```
+DI = (+DM(平滑) / TR(平滑)) × 100
-DI = (-DM(平滑) / TR(平滑)) × 100
```

### 步骤5：计算DX（方向性指数）
```
DX = (|+DI - -DI| / |+DI + -DI|) × 100
```

### 步骤6：计算ADX
```
ADX = DX在n个周期上的平滑平均值
```

## 参数

- **optInTimePeriod**（默认：14）：ADX计算的周期
  - 有效范围：2 到 100000
  - 常用值：14（标准）、7（短期）、21（长期）

## 输入
- **最高价**：`double[]`
- **最低价**：`double[]`
- **收盘价**：`double[]`

## 输出
- ADX 数值：`double[]`（范围：0 到 100）

## 解释

### ADX 水平（趋势强度）
- **0-25**：无趋势或弱趋势（横盘市场）
- **25-50**：强趋势
- **50-75**：非常强趋势
- **75-100**：极强趋势

### 常见阈值
- **ADX < 20**：非趋势，震荡，横盘市场
- **ADX 20-25**：趋势发展
- **ADX > 25**：趋势市场（阈值因市场而异）
- **ADX > 40**：非常强趋势
- **ADX > 50**：极强趋势（罕见）

### ADX 方向
- **上升ADX**：趋势加强（无论方向）
- **下降ADX**：趋势减弱或市场横盘
- **平坦ADX**：趋势强度稳定

### 关键洞察
ADX 不指示趋势方向，只指示强度：
- 高ADX可以出现在上升趋势或下降趋势中
- 使用+DI和-DI来确定方向
- ADX纯粹关注趋势强度

## 使用示例

```c

```

## 实现细节

TA-Lib ADX 实现：

1. **方向性移动**：计算+DM和-DM
2. **真实区间**：计算TR进行标准化
3. **Wilder平滑**：对DM和TR应用特殊平滑
4. **DI计算**：计算方向性指标
5. **DX计算**：推导方向性指数
6. **ADX平滑**：ADX的最终平滑

**回望周期**：ADX由于多个平滑步骤需要大量数据。

## 交易策略

### 1. 趋势过滤器策略
- **入场**：只有当ADX > 25时才交易（确认趋势）
- **方向**：使用+DI vs -DI确定方向
- **出场**：当ADX跌破20时
- **最适合**：与趋势跟踪系统结合

### 2. ADX交叉策略
- **买入**：ADX向上穿越20或25（趋势出现）
- **卖出**：ADX向下穿越20或25（趋势死亡）
- **确认**：使用+DI和-DI确定方向
- **最适合**：捕捉新兴趋势

### 3. DI交叉与ADX过滤器
- **买入**：+DI向上穿越-DI且ADX > 25
- **卖出**：-DI向上穿越+DI且ADX > 25
- **出场**：DI交叉或ADX < 20
- **最适合**：趋势市场

### 4. 极端ADX（衰竭）
- **设置**：ADX达到非常高的水平（>50）
- **信号**：ADX从极端开始下降
- **行动**：考虑获利了结
- **原因**：趋势可能正在衰竭
- **最适合**：强趋势结束

### 5. 低ADX区间交易
- **设置**：ADX < 20持续一段时间
- **策略**：使用振荡器（RSI、随机指标）
- **交易**：买入支撑，卖出阻力
- **出场**：当ADX开始上升超过20时
- **最适合**：横盘、盘整市场

## ADX 模式

### 1. 上升ADX
- **模式**：ADX向上移动
- **含义**：趋势获得强度
- **行动**：保持趋势，增加头寸
- **确认**：+DI或-DI也在上升

### 2. 下降ADX（从高水平）
- **模式**：ADX从>40下降
- **含义**：趋势失去动力
- **行动**：收紧止损，考虑获利了结
- **警告**：潜在反转或盘整

### 3. 低、平坦ADX
- **模式**：ADX < 20，无趋势
- **含义**：无趋势，横盘市场
- **行动**：避免趋势跟踪策略
- **替代**：使用区间交易策略

### 4. ADX峰值
- **模式**：ADX创高点然后转跌
- **含义**：趋势衰竭可能
- **行动**：观察反转信号
- **注意**：可能跟随盘整或反转

## 优势

1. **趋势强度量化**：趋势强度的客观衡量
2. **通用**：适用于所有市场和时间框架
3. **过滤假信号**：帮助避免震荡、无趋势条件
4. **非方向性**：关注趋势强度，而非方向
5. **久经考验**：经过时间考验，广泛使用
6. **多功能**：适用于多种策略类型

## 限制和考虑

1. **滞后指标**：多重平滑产生显著滞后
2. **无方向信息**：不告诉你趋势方向
3. **复杂计算**：比简单指标更复杂
4. **高回望**：需要大量数据来稳定
5. **水平主观性**：最佳阈值因市场而异
6. **极端罕见**：非常高的ADX（>50）很罕见

## +DI 和 -DI 使用

虽然ADX衡量强度，但+DI和-DI（通过PLUS_DI和MINUS_DI函数获得）显示方向：

### 方向性信号
- **+DI > -DI**：向上方向性移动更强
- **-DI > +DI**：向下方向性移动更强
- **+DI向上穿越-DI**：潜在买入信号（ADX确认）
- **-DI向上穿越+DI**：潜在卖出信号（ADX确认）

### 组合ADX + DI策略
```
强买入：ADX > 25 且 +DI > -DI 且 ADX上升
强卖出：ADX > 25 且 -DI > +DI 且 ADX上升
```

## 参数选择

### 周期选择
- **7周期**：更敏感，更快信号，更多噪音
- **14周期**：标准，平衡（Wilder原始）
- **21周期**：更平滑，更慢信号，更少噪音

### 阈值选择
"ADX > 25"规则不是通用的：
- **波动市场**（如加密货币）：可能使用30或35
- **稳定市场**（如债券）：可能使用20
- **股票**：通常25
- **外汇**：通常20-25

## 附加说明

J. Welles Wilder在1978年的书中与RSI、ATR和抛物线SAR一起介绍了ADX。ADX仍然是评估趋势强度最重要的指标之一。

### 关键洞察

1. **趋势与区间识别**：
   - 大多数交易者在横盘市场中亏损
   - ADX帮助识别市场何时趋势（可交易）
   - 与横盘（难以用趋势系统交易）对比
   - 仅这一点就使ADX极其有价值

2. **ADX悖论**：
   - 当ADX非常高时，趋势很强但
   - 也可能接近衰竭
   - 需要经验来解释
   - 与价格行为一起使用效果最佳

3. **Wilder平滑**：
   - 特殊平滑方法（不是EMA或SMA）
   - 产生更多滞后但更平滑的结果
   - 更适合趋势识别
   - 不太容易产生震荡

4. **多时间框架分析**：
   - 在更高时间框架检查ADX趋势
   - 使用更低时间框架进行入场时机
   - 例如，日线ADX > 25，在小时线回调时入场

5. **市场背景**：
   - 某些市场比其他市场更有趋势
   - 相应调整ADX阈值
   - 加密货币往往更经常有高ADX
   - 传统股票可能趋势较少

### 实际应用

**趋势交易者**：
- 只有当ADX > 25时才交易
- 在+DI vs -DI的方向入场
- 在ADX上升时保持交易
- 当ADX跌破20时出场

**区间交易者**：
- 当ADX < 20时交易
- 使用振荡器和支撑/阻力
- 当ADX开始上升时出场
- 等待下一个低ADX期间

**头寸规模**：
- 当ADX > 40时更大头寸（强趋势）
- 当ADX 25-40时较小头寸（中等趋势）
- 当ADX < 25时最小头寸（弱/无趋势）

ADX最好用作过滤器而不是信号生成器。它告诉你何时交易（趋势与横盘）和如何积极交易（趋势强度），但不一定告诉你哪个方向（使用+DI/-DI或其他指标）。
