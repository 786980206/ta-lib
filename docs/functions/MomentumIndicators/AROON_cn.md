# AROON - Aroon指标

## 描述

Aroon指标由Tushar Chande开发，是一个趋势跟踪指标，用于衡量给定周期内最高价和最低价出现以来的时间。它由两条线组成：Aroon Up和Aroon Down，在0到100之间振荡。Aroon通过测量最高价和最低价出现以来的时间长度来帮助识别趋势变化和强度。

## 分类
动量指标

## 作者
Tushar Chande

## 计算方法

Aroon由两个组成部分：

### Aroon Up
```
Aroon Up = ((n - 自最高价以来的周期数) / n) × 100
```

### Aroon Down
```
Aroon Down = ((n - 自最低价以来的周期数) / n) × 100
```

其中：
- n = 回望周期
- 自最高价以来的周期数 = 自最高价以来的周期数
- 自最低价以来的周期数 = 自最低价以来的周期数

### 示例
对于14周期Aroon：
- 如果最高价是3个周期前：Aroon Up = ((14-3)/14) × 100 = 78.6
- 如果最低价是1个周期前：Aroon Down = ((14-1)/14) × 100 = 92.9

## 参数

- **optInTimePeriod**（默认：14）：回望周期
  - 有效范围：2 到 100000
  - 常用值：14、25、30

## 输入
- **最高价**：`double[]`
- **最低价**：`double[]`

## 输出
- **outAroonDown**：Aroon Down数值：`double[]`（0-100）
- **outAroonUp**：Aroon Up数值：`double[]`（0-100）

## 解释

### Aroon 数值
- **100**：刚刚出现新高/低（强趋势）
- **0**：新高/低出现在n个周期前（弱趋势）
- **50**：新高/低出现在n/2个周期前（中等趋势）

### 趋势识别
1. **强上升趋势**：Aroon Up > 70，Aroon Down < 30
2. **强下降趋势**：Aroon Down > 70，Aroon Up < 30
3. **横盘**：两个Aroon值都在30-70之间
4. **趋势变化**：Aroon线交叉

### 交易信号

1. **Aroon交叉**：
   - **买入**：Aroon Up向上穿越Aroon Down
   - **卖出**：Aroon Down向上穿越Aroon Up
   - **最适合**：趋势变化检测

2. **极值**：
   - **Aroon Up = 100**：刚刚创新高（看涨）
   - **Aroon Down = 100**：刚刚创新低（看跌）
   - **两者都 = 100**：新高和新低（盘整）

3. **趋势强度**：
   - **强趋势**：一个Aroon > 70，另一个 < 30
   - **弱趋势**：两个Aroon值都在30-70
   - **无趋势**：两个Aroon值都 < 30

```c
// C/C++ 示例
```

## 实现细节

TA-Lib Aroon 实现：

1. **高低跟踪**：找到周期内的最高价和最低价
2. **周期计算**：计算自极值以来的周期数
3. **Aroon计算**：对两个组成部分应用公式
4. **回望**：需要n个周期才能产生第一个输出

## 交易策略

### 1. Aroon交叉策略
- **买入**：Aroon Up向上穿越Aroon Down
- **卖出**：Aroon Down向上穿越Aroon Up
- **过滤器**：只有当其中一个Aroon > 70时才交易
- **最适合**：趋势变化检测

### 2. Aroon + 趋势过滤器
- **过滤器**：使用ADX进行趋势强度
- **入场**：只有当ADX > 25时才使用Aroon信号
- **出场**：Aroon反转或ADX < 20
- **最适合**：避免横盘市场

### 3. Aroon振荡器策略
- **设置**：计算Aroon振荡器（Aroon Up - Aroon Down）
- **买入**：振荡器向上穿越0
- **卖出**：振荡器向下穿越0
- **最适合**：简化的Aroon信号

### 4. 极端Aroon策略
- **买入**：Aroon Up = 100（新高）
- **卖出**：Aroon Down = 100（新低）
- **确认**：等待价格确认
- **最适合**：突破交易

## Aroon振荡器

Aroon振荡器是Aroon Up和Aroon Down之间的差值：

```
Aroon振荡器 = Aroon Up - Aroon Down
```

- **正值**：Aroon Up > Aroon Down（上升趋势）
- **负值**：Aroon Down > Aroon Up（下降趋势）
- **零**：相等的Aroon值（横盘）

## 优势

1. **趋势识别**：非常适合趋势检测
2. **早期信号**：提供早期趋势变化信号
3. **清晰水平**：0-100刻度直观
4. **双组成部分**：两条线提供确认
5. **通用**：适用于所有市场和时间框架

## 限制

1. **滞后**：基于历史极值
2. **震荡**：可能产生假信号
3. **周期敏感性**：结果随周期选择变化
4. **无幅度**：不测量价格移动大小
5. **震荡市场**：在横盘条件下困难

## 周期选择

### 短期（5-10）
- **特征**：更敏感，频繁信号
- **用途**：日内交易，短期趋势
- **权衡**：更多信号，更多噪音
- **最适合**：主动交易

### 标准期（14-25）
- **特征**：平衡敏感性
- **用途**：摆动交易，一般分析
- **权衡**：良好平衡
- **最适合**：大多数交易风格

### 长期（30-50）
- **特征**：不太敏感，更少信号
- **用途**：头寸交易，长期趋势
- **权衡**：更少信号，更可靠
- **最适合**：长期分析

## 相关函数

- **ADX**：平均方向性指数 - 趋势强度
- **PLUS_DI**：正方向性指标 - 向上移动
- **MINUS_DI**：负方向性指标 - 向下移动

## 附加说明

Tushar Chande开发了Aroon作为他趋势跟踪系统工作的一部分。关键洞察是自上次极值（高点或低点）以来的时间比价格移动的幅度对趋势识别更重要。

### 关键洞察

1. **基于时间的分析**：
   - 关注极值何时出现
   - 不是价格移动多少
   - 时间衰减概念
   - 趋势分析的独特方法

2. **趋势强度测量**：
   - Aroon Up = 100：强上升趋势（最近高点）
   - Aroon Down = 100：强下降趋势（最近低点）
   - 两者都高：盘整（最近高点和低点）
   - 两者都低：弱趋势（旧极值）

3. **早期预警系统**：
   - Aroon在价格极值之前变化
   - 可以信号趋势衰竭
   - 对出场时机有用
   - 补充基于价格的指标

4. **最佳应用**：
   - 趋势识别
   - 趋势变化检测
   - 趋势强度评估
   - 突破确认

5. **组合策略**：
   - 使用Aroon进行趋势方向
   - 使用价格行为进行入场时机
   - 使用成交量确认
   - 使用多时间框架背景

### 实用技巧

**趋势跟踪**：
- 等待Aroon Up > 70进行上升趋势
- 等待Aroon Down > 70进行下降趋势
- 在趋势确认时在回调时入场
- 当Aroon线交叉时出场

**趋势变化**：
- 观察Aroon交叉
- 用价格行为确认
- 使用成交量验证
- 在最近极值之外设置止损

**区间交易**：
- 当两个Aroon值都 < 30时避免
- 使用振荡器代替
- 等待Aroon显示趋势
- 当Aroon确认时交易突破

Aroon对于识别趋势变化和测量趋势强度特别有价值。它是少数几个关注时间而不是价格幅度的指标之一，为市场行为提供了独特的视角。
