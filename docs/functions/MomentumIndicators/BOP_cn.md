# BOP - 买卖力量平衡

## 概述
买卖力量平衡（BOP）通过评估买卖双方将价格推向极端水平的能力来衡量买方与卖方的力量。它使用收盘价、开盘价、最高价和最低价之间的关系计算。

## 分类
动量指标

## 计算方法
BOP函数计算买卖压力之间的平衡：

### 公式
```
BOP = (收盘价 - 开盘价) / (最高价 - 最低价)
```

其中：
- **收盘价 - 开盘价**：衡量收盘相对于开盘的强度
- **最高价 - 最低价**：代表价格移动的总范围

## 参数
- **输入**：OHLC数据（开盘价、最高价、最低价、收盘价）
- **输出**：BOP数组（数值范围从-1到+1）

## 含义
BOP指标显示：
- **接近+1的值**：强买入压力（收盘价接近最高价）
- **接近-1的值**：强卖出压力（收盘价接近最低价）
- **接近0的值**：买卖双方平衡
- **趋势**：正值表示看涨压力，负值表示看跌压力

## 使用方法
1. **趋势确认**：用于确认趋势的强度
2. **背离**：寻找价格和BOP之间的背离
3. **超买/超卖**：极值可能表示潜在反转
4. **零线**：零线穿越可以信号趋势变化

## 使用示例
```c
#include "ta_libc.h"

int main() {
    // 计算BOP
    TA_RetCode ret = TA_BOP(
        0,                    // startIdx
        99,                   // endIdx
        open_prices,          // open
        high_prices,         // high
        low_prices,          // low
        close_prices,        // close
        &outBegIdx,          // outBegIdx
        &outNBElement,       // outNBElement
        bop                  // outReal
    );
    
    if (ret == TA_SUCCESS) {
        // 使用BOP数组进行分析
        for (int i = 0; i < outNBElement; i++) {
            if (bop[i] > 0.5) {
                // 强买入压力
            } else if (bop[i] < -0.5) {
                // 强卖出压力
            }
        }
    }
}
```

## 交易策略
1. **趋势跟踪**：在BOP方向进入交易
2. **背离交易**：交易价格和BOP之间的背离
3. **零线交叉**：当BOP穿越零时进入头寸
4. **极值**：在极端BOP值寻找反转

## 优势
- 简单直观的解释
- 对日内交易效果很好
- 可以识别买卖压力
- 对确认趋势有用

## 限制
- 在震荡市场中可能给出假信号
- 最好与其他指标一起使用
- 在低波动性期间可能噪音较大
- 不考虑成交量
