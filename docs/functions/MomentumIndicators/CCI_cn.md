# CCI - 商品通道指数

## 描述

商品通道指数（CCI）是一个多功能指标，可用于识别周期性趋势、超买/超卖水平和背离。最初为商品开发，CCI在股票、指数和其他证券上同样有效。它测量价格相对于其统计均值的偏差。

## 分类
动量指标

## 作者
Donald Lambert

## 计算方法

CCI测量价格相对于正常偏差从其平均价格偏离的程度：

### 公式
```
CCI = (典型价格 - 典型价格的SMA) / (0.015 × 平均偏差)
```

其中：
- **典型价格** = (最高价 + 最低价 + 收盘价) / 3
- **SMA** = 典型价格在周期上的简单移动平均线
- **平均偏差** = 从SMA的绝对偏差的平均值

### 详细步骤
```
步骤1：计算每个周期的典型价格
        TP = (H + L + C) / 3

步骤2：计算典型价格的SMA
        SMA_TP = SMA(TP, 周期)

步骤3：计算平均偏差
        MD = SMA(|TP - SMA_TP|, 周期)

步骤4：计算CCI
        CCI = (TP - SMA_TP) / (0.015 × MD)
```

常数0.015确保约70-80%的CCI值落在-100和+100之间。

## 参数

- **optInTimePeriod**（默认：14）：计算SMA和平均偏差的周期
  - 有效范围：2 到 100000
  - 常用值：14（标准）、20、30

## 输入
- **最高价**：`double[]`
- **最低价**：`double[]`
- **收盘价**：`double[]`

## 输出
- CCI 数值：`double[]`（通常范围：-300到+300，但无界）

## 解释

### 超买/超卖水平
- **高于+100**：超买（价格相对于平均值异常高）
- **低于-100**：超卖（价格相对于平均值异常低）
- **-100和+100之间**：正常范围

### 关键水平
- **+200**：极度超买
- **+100**：超买阈值
- **0**：中心线（平衡）
- **-100**：超卖阈值
- **-200**：极度超卖

### 交易信号

1. **超买/超卖**：
   - **买入**：CCI从下方向上穿越-100（超卖恢复）
   - **卖出**：CCI从上方向下穿越+100（超买下降）
   - 与趋势确认更可靠

2. **中心线交叉**：
   - **看涨**：CCI向上穿越0（获得动量）
   - **看跌**：CCI向下穿越0（失去动量）
   - 表示动量转换

3. **极端水平**：
   - **买入**：CCI移动到-200以下然后恢复
   - **卖出**：CCI移动到+200以上然后下降
   - 表示极度衰竭

4. **背离**：
   - **看涨**：价格更低低点，CCI更高低点
   - **看跌**：价格更高高点，CCI更低高点
   - 信号潜在趋势反转

5. **趋势识别**：
   - **强上升趋势**：CCI持续高于0
   - **强下降趋势**：CCI持续低于0
   - **弱趋势**：CCI在0附近振荡

## 使用示例

```c
// C/C++ 示例
double high[100], low[100], close[100];
double cciOutput[100];
int outBegIdx, outNBElement;

// 计算14周期CCI
TA_RetCode retCode = TA_CCI(
    0,                    // 开始索引
    99,                   // 结束索引
    high,                 // 最高价
    low,                  // 最低价
    close,                // 收盘价
    14,                   // 时间周期
    &outBegIdx,           // 输出：开始索引
    &outNBElement,        // 输出：元素数量
    cciOutput             // 输出：CCI数值
);
```

## 实现细节

TA-Lib CCI 实现：

1. **典型价格**：计算最高价、最低价、收盘价的平均值
2. **SMA计算**：计算简单移动平均线
3. **平均偏差**：计算平均绝对偏差
4. **缩放**：应用0.015常数进行标准化
5. **无界输出**：CCI在极端情况下可以超过±300

### 计算步骤
```
步骤1：对每个柱，TP = (H + L + C) / 3
步骤2：计算周期上TP的SMA
步骤3：计算从SMA的平均偏差
步骤4：CCI = (TP - SMA_TP) / (0.015 × 平均偏差)
```

## 交易策略

### 1. 经典超买/超卖
- **买入**：CCI < -100，然后向上穿越-100
- **卖出**：CCI > +100，然后向下穿越+100
- **止损**：最近摆动低点/高点
- **目标**：相反极值或中心线
- **最适合**：区间市场

### 2. CCI趋势交易
- **上升趋势入场**：在上升趋势中CCI跌至-100时买入
- **下降趋势入场**：在下降趋势中CCI升至+100时卖出
- **过滤器**：使用MA或趋势线确认趋势
- **出场**：相反CCI信号或趋势突破
- **最适合**：趋势市场

### 3. 背离策略
- **设置**：识别CCI与价格的背离
- **确认**：等待价格模式（双顶/底）
- **入场**：模式完成时
- **止损**：模式极值之外
- **目标**：1:2或1:3风险/回报
- **最适合**：趋势衰竭点

### 4. 零线策略
- **买入**：CCI向上穿越0
- **卖出**：CCI向下穿越0
- **确认**：可以等待收盘高于/低于0
- **出场**：相反信号
- **最适合**：更短时间框架，动量交易

### 5. 极端CCI（>200或<-200）
- **均值回归**：淡化极端水平
- **入场**：当CCI开始返回正常时
- **止损**：最近极值之外
- **目标**：返回±100或0
- **最适合**：有均值回归的波动市场

## 周期选择

### 不同周期
- **短期（5-10）**：更敏感，频繁信号
  - 日内交易
  - 波动市场
  - 更多假信号

- **标准（14-20）**：平衡方法
  - 摆动交易
  - 大多数市场
  - Lambert原始

- **长期（30-50）**：更平滑，更少信号
  - 头寸交易
  - 不太波动的市场
  - 更少假信号

## 优势

1. **多功能**：在趋势和区间市场中有效
2. **清晰水平**：定义的超买/超卖区域
3. **无界**：可以信号极强的移动（>±200）
4. **领先成分**：可以领先价格转向
5. **通用**：在所有市场中都有效
6. **多种用途**：超买/超卖、趋势、背离

## 限制和考虑

1. **趋势中的假信号**：在强趋势期间可以保持超买/超卖
2. **震荡**：在震荡市场中产生假信号
3. **无成交量**：不包含成交量数据
4. **主观水平**：最佳阈值因市场而异
5. **周期敏感性**：结果随周期选择变化
6. **滞后元素**：使用SMA，有滞后

## CCI模式和行为

### 在趋势市场中
- **强上升趋势**：CCI经常保持在+100以上
- **强下降趋势**：CCI经常保持在-100以下
- **不要淡化趋势**：超买可以保持超买
- **用于入场**：在上升趋势中买入跌至-100

### 在区间市场中
- **规则振荡**：CCI在±100之间移动
- **均值回归**：极值倾向于反转
- **最佳CCI应用**：交易区间极值
- **清晰信号**：更可靠的超买/超卖

### 极端读数
- **高于+200**：异常强动量
  - 可以继续或信号衰竭
  - 观察背离
  - 考虑获利了结

- **低于-200**：异常弱动量
  - 可以继续或信号投降
  - 观察反转信号
  - 考虑反向入场

## 相关函数

- **RSI**：相对强弱指数 - 另一个超买/超卖振荡器
- **STOCH**：随机振荡器 - 类似的动量工具
- **WILLR**：威廉%R - 类似的基于区间的振荡器
- **MOM**：动量 - 更简单的动量指标

## 附加说明

Donald Lambert在1980年引入了CCI，最初设计用于识别商品的周期性转向。尽管其名称，它在所有证券类型上都有效。

### 关键洞察

1. **0.015常数**：
   - 选择确保约70-80%的值在±100之间
   - 使±100在统计上显著
   - ±100之外的值代表异常价格行为
   - 不是硬限制 - 值可以超过±300

2. **与RSI或随机指标不同**：
   - CCI无界（无0-100限制）
   - 可以显示超出标准水平的极端强度/弱点
   - 这是特性，不是缺陷
   - 极端读数信号异常条件

3. **典型价格使用**：
   - 使用(H+L+C)/3而不是仅收盘价
   - 包含完整价格区间
   - 比仅收盘价指标更全面
   - 更好地捕获日内波动性

4. **均值回归与趋势**：
   - 区间市场：CCI均值回归（±100可靠）
   - 趋势市场：CCI可以保持延伸（±100不太可靠）
   - 关键是识别市场类型
   - 与趋势指标结合（ADX、MA）

5. **多时间框架分析**：
   - 更高时间框架用于趋势背景
   - 更低时间框架用于入场时机
   - 例如，日线CCI用于趋势，小时线用于入场
   - 两者应该一致以获得最佳交易

### 实用技巧

**区间交易**：
- 严格使用CCI ±100水平
- 在返回正常范围时入场
- 在最近高点/低点之外设置紧止损
- 目标相反极值

**趋势交易**：
- 在上升趋势中忽略超买
- 在下降趋势中忽略超卖
- 使用CCI极值作为入场点
- 仅在趋势方向交易

**所有交易**：
- CCI > +100 = 看涨偏向
- CCI < -100 = 看跌偏向
- CCI在0附近振荡 = 无明确趋势
- 背离比绝对水平更可靠

**背离交易**：
- 最强大的CCI信号
- 等待确认（支撑/阻力）
- 在主要高点/低点最佳
- 不要交易每个背离

CCI在用作多功能工具时特别有效：趋势过滤器（中心线）、动量测量（绝对水平）和反转指标（背离）。关键是适应其解释以适应当前市场条件。
