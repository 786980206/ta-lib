# CDLTASUKIGAP - 带子跳空

## 概述
带子跳空是一种延续形态，其中跳空后跟一根回调蜡烛，但不关闭跳空。

## 分类
形态识别

## 计算方法
CDLTASUKIGAP 函数识别带子跳空形态：

### 向上带子跳空判断标准
- 向上跳空（实体之间）
- 跳空后第一根蜡烛：白（看涨）蜡烛
- 跳空后第二根蜡烛：在前一实体内开盘但不关闭跳空的黑蜡烛

### 向下带子跳空判断标准
- 向下跳空（实体之间）
- 跳空后第一根蜡烛：黑（看跌）蜡烛
- 跳空后第二根蜡烛：在前一实体内开盘但不关闭跳空的白蜡烛

## 实现说明

**此形态使用TA-Lib的自适应阈值系统**来确定蜡烛特征，如"长实体"、"短实体"、"十字星"等。这些阈值会根据最近的市场行为自动调整，而不是使用固定百分比。

## 参数
- **输入**：OHLC数据（开盘价、最高价、最低价、收盘价）
- **输出**：形态识别数组（100表示看涨，-100表示看跌，0表示无形态）

## 含义
带子跳空形态表明：
- **看涨信号**：潜在的向上运动
- **看跌信号**：潜在的向下运动
- **市场情绪**：显示当前市场心理
- **交易机会**：提供入场/出场信号

## 使用方法

## 使用示例
```c
#include "ta_libc.h"

int main() {
    // 计算CDLTASUKIGAP
    TA_RetCode ret = TA_CDLTASUKIGAP(
        0,                    // startIdx
        99,                   // endIdx
        open_prices,          // open
        high_prices,         // high
        low_prices,          // low
        close_prices,        // close
        &outBegIdx,          // outBegIdx
        &outNBElement,       // outNBElement
        pattern              // outInteger
    );
    
    if (ret == TA_SUCCESS) {
        for (int i = 0; i < outNBElement; i++) {
            if (pattern[i] == 100) {
                // 看涨形态
            } else if (pattern[i] == -100) {
                // 看跌形态
            }
        }
    }
}
```
