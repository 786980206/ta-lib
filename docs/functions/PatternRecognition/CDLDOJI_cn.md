# CDLDOJI - 十字星（Doji）

## 描述

十字星（Doji）是一种K线形态，当开盘价和收盘价非常接近或相等时出现，形成类似十字的形状。十字星代表市场犹豫不决，可能预示着潜在的趋势反转，特别是在强势趋势后出现时。它是技术分析中最重要的单根K线形态之一。

## 分类
形态识别 / K线形态

## 作者
传统日本K线分析

## 计算方法

当开盘价和收盘价非常接近时识别十字星：

### 判断标准
当实体（开盘价和收盘价之间的差值）相对于总波动范围非常小时，检测到十字星。确切的阈值是自适应的，会根据最近的市场行为进行调整。

**概念上**：|收盘价 - 开盘价| 相对于（最高价 - 最低价）非常小

### 视觉特征
- **小实体**：非常小的实体或无实体
- **长影线**：上影线和下影线可能很长
- **十字形状**：类似十字或加号
- **犹豫不决**：代表市场不确定性

## 实现说明

**此形态使用TA-Lib的自适应阈值系统**来确定蜡烛特征，如"长实体"、"短实体"、"十字星"等。这些阈值会根据最近的市场行为自动调整，而不是使用固定百分比。

## 参数

无 - CDLDOJI 是形态识别函数

## 输入
- **开盘价**：`double[]`
- **最高价**：`double[]`
- **最低价**：`double[]`
- **收盘价**：`double[]`

## 输出
- 形态值：`int[]`（100 = 十字星，0 = 非十字星）

## 解读

### 十字星类型

1. **标准十字星**：
   - 开盘价 ≈ 收盘价
   - 有上影线和下影线
   - 最常见类型

2. **长腿十字星**：
   - 开盘价 ≈ 收盘价
   - 非常长的上影线和下影线
   - 高波动性，强烈犹豫

3. **蜻蜓十字星**：
   - 开盘价 ≈ 收盘价
   - 长下影线，无上影线
   - 看涨反转信号

4. **墓碑十字星**：
   - 开盘价 ≈ 收盘价
   - 长上影线，无下影线
   - 看跌反转信号

### 交易信号

1. **反转信号**：
   - **看涨**：下跌趋势后的十字星
   - **看跌**：上涨趋势后的十字星
   - **确认**：下一根K线确认方向

2. **犹豫信号**：
   - 市场不确定性
   - 潜在趋势变化
   - 等待确认
   - 成交量分析重要

3. **支撑/阻力**：
   - 关键位置的十字星
   - 强反转潜力
   - 多时间框架分析
   - 价格行为确认

## 使用示例

```c
// C/C++ 示例
double open[100], high[100], low[100], close[100];
int dojiPattern[100];
int outBegIdx, outNBElement;

// 检测十字星形态
TA_RetCode retCode = TA_CDLDOJI(
    0,                    // 开始索引
    99,                   // 结束索引
    open,                 // 开盘价
    high,                 // 最高价
    low,                  // 最低价
    close,                // 收盘价
    &outBegIdx,           // 输出：开始索引
    &outNBElement,        // 输出：元素数量
    dojiPattern           // 输出：十字星形态
);
```

## 实现细节

TA-Lib CDLDOJI 实现：

1. **实体计算**：确定实体是否足够小
2. **阈值检查**：应用10%阈值规则
3. **形态识别**：返回100表示十字星，否则返回0
4. **无回看**：独立分析每根蜡烛

## 交易策略

### 1. 十字星反转策略
- **设置**：强势趋势后的十字星
- **入场**：等待确认K线
- **方向**：与前期趋势相反
- **止损**：超越十字星范围
- **最佳适用**：趋势衰竭

### 2. 十字星 + 支撑/阻力
- **设置**：关键位置的十字星
- **入场**：十字星突破
- **确认**：成交量和价格行为
- **最佳适用**：关键技术位

### 3. 十字星 + 成交量分析
- **设置**：高成交量的十字星
- **入场**：成交量确认
- **方向**：成交量方向
- **最佳适用**：高确信度设置

### 4. 多重十字星策略
- **设置**：连续多个十字星
- **入场**：明确方向
- **确认**：强势价格行为
- **最佳适用**：盘整期

## 十字星变体

### 1. 长腿十字星
- **特征**：非常长的影线
- **含义**：高波动性，强烈犹豫
- **信号**：强反转潜力
- **最佳适用**：高波动性市场

### 2. 蜻蜓十字星
- **特征**：长下影线，无上影线
- **含义**：卖方被拒绝，买方介入
- **信号**：看涨反转
- **最佳适用**：下跌趋势底部

### 3. 墓碑十字星
- **特征**：长上影线，无下影线
- **含义**：买方被拒绝，卖方介入
- **信号**：看跌反转
- **最佳适用**：上涨趋势顶部

## 优势

1. **清晰信号**：明显的犹豫形态
2. **反转潜力**：强反转信号
3. **通用性**：适用于所有市场
4. **简单**：易于识别
5. **可靠**：确认后高成功率

## 局限性

1. **需要确认**：不是独立信号
2. **虚假信号**：可能无反转发生
3. **依赖背景**：需要趋势背景
4. **成交量重要**：需要成交量确认
5. **时机**：入场时机可能困难

## 附加说明

十字星是最重要的K线形态之一，因为它代表纯粹的市场犹豫。当它在强势趋势后出现时特别重要，因为它通常预示着衰竭和潜在反转。

### 关键洞察

1. **犹豫信号**：
   - 代表市场不确定性
   - 买卖双方平衡
   - 潜在趋势变化
   - 需要确认

2. **反转潜力**：
   - 强反转信号
   - 特别是在趋势后
   - 成交量确认重要
   - 下一根K线方向关键

3. **背景重要**：
   - 上涨趋势后的十字星 = 看跌
   - 下跌趋势后的十字星 = 看涨
   - 区间内的十字星 = 犹豫
   - 极值处的十字星 = 反转

4. **需要确认**：
   - 永远不要单独交易十字星
   - 等待下一根K线
   - 使用成交量分析
   - 结合支撑/阻力

5. **最佳应用**：
   - 趋势衰竭点
   - 关键支撑/阻力位
   - 高波动期
   - 多时间框架分析

### 实用技巧

**反转交易**：
- 等待强势趋势后的十字星
- 用下一根K线确认
- 使用成交量验证
- 在十字星范围外设置止损

**支撑/阻力**：
- 关键位置的十字星 = 强信号
- 等待突破方向
- 使用成交量确认
- 在十字星外设置止损

**犹豫分析**：
- 十字星 = 市场不确定性
- 等待明确方向
- 使用其他指标
- 避免交易直到解决

**风险管理**：
- 十字星 = 高不确定性
- 减少仓位大小
- 使用更宽止损
- 等待确认

十字星对于识别潜在反转点和市场犹豫特别有价值。当正确确认并在适当背景下使用时，它是最可靠的K线形态之一。
