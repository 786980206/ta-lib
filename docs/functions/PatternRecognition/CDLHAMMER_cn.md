# CDLHAMMER - 锤子线

## 描述

锤子线是一种看涨反转K线形态，通常出现在下跌趋势的底部。它在蜡烛顶部有一个小实体，长下影线，很少或没有上影线。锤子线表明卖方将价格大幅推低，但买方介入将价格推回，形成潜在的反转信号。

## 分类
形态识别 / K线形态

## 作者
传统日本K线分析

## 计算方法

锤子线通过特定标准识别：

### 判断标准
1. **小实体**：实体在蜡烛顶部
2. **长下影线**：明显长于实体（自适应阈值）
3. **很少上影线**：最小或无上影线
4. **位置**：出现在下跌趋势后

### 视觉特征
- **小实体**：顶部小实体
- **长下影线**：长下影线
- **无上影线**：最小上影线
- **反转信号**：看涨反转形态

## 实现说明

**此形态使用TA-Lib的自适应阈值系统**来确定蜡烛特征，如"长实体"、"短实体"、"十字星"等。这些阈值会根据最近的市场行为自动调整，而不是使用固定百分比。

## 参数

无 - CDLHAMMER 是形态识别函数

## 输入
- **开盘价**：`double[]`
- **最高价**：`double[]`
- **最低价**：`double[]`
- **收盘价**：`double[]`

## 输出
- 形态值：`int[]`（100 = 锤子线，0 = 非锤子线）

## 解读

### 锤子线特征
1. **小实体**：实体相对于范围的大小
2. **长下影线**：至少是实体大小的2倍
3. **最小上影线**：很少或无上影线
4. **看涨信号**：潜在向上反转

### 交易信号

1. **看涨反转**：
   - **设置**：下跌趋势后的锤子线
   - **信号**：潜在底部形成
   - **确认**：下一根K线上涨
   - **最佳适用**：下跌趋势底部

2. **支撑位**：
   - **设置**：支撑位的锤子线
   - **信号**：强支撑确认
   - **入场**：从支撑反弹
   - **最佳适用**：关键支撑位

3. **成交量确认**：
   - **设置**：高成交量的锤子线
   - **信号**：强买入兴趣
   - **入场**：成交量确认
   - **最佳适用**：高确信度设置

## 实现细节

TA-Lib CDLHAMMER 实现：

1. **实体计算**：确定实体大小和位置
2. **影线分析**：测量下影线和上影线
3. **比例检查**：确保下影线是实体大小的2倍
4. **形态识别**：返回100表示锤子线，否则返回0

## 交易策略

### 1. 锤子线反转策略
- **设置**：下跌趋势后的锤子线
- **入场**：下一根K线上涨
- **止损**：锤子线低点下方
- **目标**：前期阻力位
- **最佳适用**：趋势反转

### 2. 锤子线 + 支撑策略
- **设置**：支撑位的锤子线
- **入场**：从支撑反弹
- **止损**：支撑下方
- **目标**：下一个阻力位
- **最佳适用**：支撑交易

### 3. 锤子线 + 成交量策略
- **设置**：高成交量的锤子线
- **入场**：成交量确认
- **止损**：锤子线低点下方
- **目标**：测量移动
- **最佳适用**：高确信度设置

### 4. 多重锤子线策略
- **设置**：连续多个锤子线
- **入场**：明确方向
- **止损**：所有锤子线低点下方
- **目标**：前期阻力位
- **最佳适用**：强支撑区域

## 锤子线变体

### 1. 标准锤子线
- **特征**：小实体，长下影线
- **含义**：买方在低点介入
- **信号**：看涨反转
- **最佳适用**：下跌趋势底部

### 2. 倒锤子线
- **特征**：小实体，长上影线
- **含义**：买方试图推高
- **信号**：潜在反转
- **最佳适用**：下跌趋势底部

### 3. 蜻蜓十字星
- **特征**：无实体，长下影线
- **含义**：强烈拒绝低点
- **信号**：非常看涨
- **最佳适用**：强支撑位

## 优势

1. **清晰信号**：明显的反转形态
2. **高成功率**：确认后可靠

5. **时机**：入场时机可能困难

## 附加说明

锤子线是最可靠的看涨反转形态之一。当它在强势下跌趋势后出现时特别重要，因为它通常预示着卖压已经衰竭，买方正在介入。

### 关键洞察

1. **反转信号**：
   - 强看涨反转形态
   - 卖方衰竭，买方介入
   - 潜在底部形成
   - 需要确认

2. **支撑确认**：
   - 支撑位的锤子线 = 强信号
   - 多个锤子线 = 非常强支撑
   - 成交量确认重要
   - 下一根K线方向关键

3. **风险管理**：
   - 清晰止损位（锤子线低点下方）
   - 风险/回报比有利
   - 仓位大小重要
   - 多时间框架分析

4. **最佳应用**：
   - 下跌趋势底部
   - 支撑位交易
   - 反转策略
   - 高确信度设置

5. **需要确认**：
   - 永远不要单独交易锤子线
   - 等待下一根K线
   - 使用成交量分析
   - 结合支撑/阻力

### 实用技巧

**反转交易**：
- 等待下跌趋势后的锤子线
- 用下一根K线上涨确认
- 使用成交量验证
- 在锤子线低点下方设置止损

**支撑交易**：
- 支撑位的锤子线 = 强信号
- 等待反弹确认
- 使用成交量验证
- 在支撑下方设置止损

**风险管理**：
- 锤子线下方的清晰止损位
- 有利的风险/回报比
- 基于风险的仓位大小
- 多时间框架确认

**入场时机**：
- 等待下一根K线上涨
- 使用成交量确认
- 结合其他指标
- 避免在震荡市场中使用

锤子线对于识别潜在反转点和强支撑位特别有价值。当正确确认并在适当背景下使用时，它是最可靠的K线形态之一。
