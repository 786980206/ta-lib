# CDL3INSIDE - 三内包形态

## 概述
三内包形态（CDL3INSIDE）是一种K线反转形态。三内包上涨是看涨的（孕线形态后跟确认），三内包下跌是看跌的（看跌孕线后跟确认）。

## 分类
形态识别

## 计算方法
CDL3INSIDE 函数通过分析三根连续蜡烛来识别三内包形态：

### 三内包上涨（看涨）判断标准
- 第一根K线：长黑（看跌）实体
- 第二根K线：短实体被第一根K线包含（孕线形态）
- 第三根K线：白（看涨）K线，收盘价高于第一根K线的开盘价

### 三内包下跌（看跌）判断标准
- 第一根K线：长白（看涨）实体
- 第二根K线：短实体被第一根K线包含（孕线形态）
- 第三根K线：黑（看跌）K线，收盘价低于第一根K线的开盘价

## 实现说明

**此形态使用TA-Lib的自适应阈值系统**来确定蜡烛特征，如"长实体"、"短实体"、"十字星"等。这些阈值会根据最近的市场行为自动调整，而不是使用固定百分比。

## 参数
- **输入**：OHLC数据（开盘价、最高价、最低价、收盘价）
- **输出**：形态识别数组（100表示看涨，-100表示看跌，0表示无形态）

## 含义
三内包形态表明：
- **看涨三内包**：潜在的向上延续或反转
- **看跌三内包**：潜在的向下延续或反转
- **盘整**：显示市场盘整
- **确认**：应通过其他指标确认

## 使用方法

## 使用示例
```c
#include "ta_libc.h"

int main() {
    // 计算CDL3INSIDE
    TA_RetCode ret = TA_CDL3INSIDE(
        0,                    // startIdx
        99,                   // endIdx
        open_prices,          // open
        high_prices,         // high
        low_prices,          // low
        close_prices,        // close
        &outBegIdx,          // outBegIdx
        &outNBElement,      // outNBElement
        pattern              // outInteger
    );
    
    if (ret == TA_SUCCESS) {
        // 使用形态数组进行分析
        for (int i = 0; i < outNBElement; i++) {
            if (pattern[i] == 100) {
                // 看涨三内包形态
            } else if (pattern[i] == -100) {
                // 看跌三内包形态
            }
        }
    }
}
```
