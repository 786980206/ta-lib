# CDLBREAKAWAY - 脱离形态

## 概述
脱离形态（CDLBREAKAWAY）是一种五根K线的反转形态，可以是看涨或看跌的。它显示一个跳空，然后是相反方向的三个小K线，接着是原始方向的跳空。

## 分类
形态识别

## 计算方法
CDLBREAKAWAY 函数识别脱离形态：

### 看涨脱离判断标准
- 第一根K线：长黑（看跌）
- 第二根K线：向下跳空的黑实体
- 第三根K线：黑或白，高点低点和低点都低于第二根
- 第四根K线：黑或白，高点低点和低点都低于第三根
- 第五根K线：白（看涨），收盘价在跳空内，抹去前三天的走势

### 看跌脱离判断标准
- 第一根K线：长白（看涨）
- 第二根K线：向上跳空的白实体
- 第三根K线：黑或白，高点高点和低点都高于第二根
- 第四根K线：黑或白，高点高点和低点都高于第三根
- 第五根K线：黑（看跌），收盘价在跳空内，抹去前三天的走势
- 显示趋势反转

## 参数
- **输入**：OHLC数据（开盘价、最高价、最低价、收盘价）
- **输出**：形态识别数组（100表示看涨，-100表示看跌，0表示无形态）

## 含义
脱离形态表明：
- **看涨脱离**：盘整后的强向上反转
- **看跌脱离**：盘整后的强向下反转
- **趋势反转**：显示清晰的趋势变化
- **高可靠性**：强反转信号

## 使用方法
1. **反转信号**：在与前期趋势相反的方向入场
2. **趋势变化**：寻找清晰的趋势反转
3. **成交量确认**：更高的成交量强化信号
4. **仓位大小**：鉴于可靠性增加仓位大小

## 使用示例
```c
#include "ta_libc.h"

int main() {
    // 计算CDLBREAKAWAY
    TA_RetCode ret = TA_CDLBREAKAWAY(
        0,                    // startIdx
        99,                   // endIdx
        open_prices,          // open
        high_prices,         // high
        low_prices,          // low
        close_prices,        // close
        &outBegIdx,          // outBegIdx
        &outNBElement,       // outNBElement
        pattern              // outInteger
    );
    
    if (ret == TA_SUCCESS) {
        for (int i = 0; i < outNBElement; i++) {
            if (pattern[i] == 100) {
                // 看涨脱离
            } else if (pattern[i] == -100) {
                // 看跌脱离
            }
        }
    }
}
```

## 优势
- 高度可靠的反转形态
- 清晰的趋势变化信号
- 适用于所有时间框架
- 强劲的历史记录

## 局限性
- 需要六根K线完成
- 相对罕见的形态
- 没有确认可能有虚假信号
- 在重要位置最有效
