# CDLPIERCING - 刺透形态

## 概述
刺透形态（CDLPIERCING）是一种K线形态，由两根蜡烛组成，其中第二根蜡烛在第一根蜡烛的低点下方开盘，收盘价高于第一根蜡烛实体的中点。这是一种看涨反转形态，可以发出趋势变化信号。

## 分类
形态识别

## 计算方法
CDLPIERCING 函数通过比较当前和前一蜡烛来识别刺透形态：

### 刺透形态判断标准
- 前一蜡烛是看跌的（收盘价 < 开盘价）
- 当前蜡烛是看涨的（收盘价 > 开盘价）
- 当前蜡烛在前一蜡烛低点下方开盘
- 当前蜡烛收盘价高于前一蜡烛实体的中点
- 当前蜡烛收盘价低于前一蜡烛的开盘价

### 公式
```
前一实体中点 = (前一开盘价 + 前一收盘价) / 2
当前收盘价 > 前一实体中点
当前开盘价 < 前一低点
当前收盘价 < 前一开盘价
```

## 实现说明

**此形态使用TA-Lib的自适应阈值系统**来确定蜡烛特征，如"长实体"、"短实体"、"十字星"等。这些阈值会根据最近的市场行为自动调整，而不是使用固定百分比。

## 参数
- **输入**：OHLC数据（开盘价、最高价、最低价、收盘价）
- **输出**：形态识别数组（100表示刺透形态，0表示无形态）

## 含义
刺透形态表明：
- **看涨反转**：下跌趋势后潜在的向上反转
- **买盘压力**：显示买盘兴趣增加
- **支撑测试**：经常出现在关键支撑位
- **确认**：应该被其他指标确认

## 使用方法
1. **反转信号**：在趋势极值寻找刺透形态
2. **成交量确认**：用成交量分析确认
3. **支撑/阻力**：在关键支撑/阻力位使用
4. **趋势背景**：考虑整体趋势方向

## 使用示例
```c
#include "ta_libc.h"

int main() {
    // 计算CDLPIERCING
    TA_RetCode ret = TA_CDLPIERCING(
        0,                    // startIdx
        99,                   // endIdx
        open_prices,          // open
        high_prices,         // high
        low_prices,          // low
        close_prices,        // close
        &outBegIdx,          // outBegIdx
        &outNBElement,      // outNBElement
        pattern              // outInteger
    );
    
    if (ret == TA_SUCCESS) {
        // 使用形态数组进行分析
        for (int i = 0; i < outNBElement; i++) {
            if (pattern[i] == 100) {
                // 刺透形态
            }
        }
    }
}
```

## 优势
- 显示潜在的看涨反转
- 视觉上易于识别
- 适用于所有时间框架
- 可与其他指标结合使用

## 局限性
- 在震荡市场中可能给出虚假信号
- 需要其他指标确认
- 在关键支撑/阻力位效果最好
- 在快速移动市场中可能滞后
