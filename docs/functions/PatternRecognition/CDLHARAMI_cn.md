# CDLHARAMI - 孕线形态

## 概述
孕线形态（CDLHARAMI）是一种K线形态，由两根蜡烛组成，其中第二根蜡烛的实体完全包含在第一根蜡烛的实体内。这是一个反转形态，可以预示趋势变化。

## 分类
形态识别

## 计算方法
CDLHARAMI 函数通过比较当前和前期蜡烛来识别孕线形态：

### 看涨孕线
- 前期蜡烛是看跌的（收盘价 < 开盘价）
- 当前蜡烛是看涨的（收盘价 > 开盘价）
- 当前蜡烛的实体完全包含在前期蜡烛的实体内
- 当前开盘价 > 前期收盘价
- 当前收盘价 < 前期开盘价

### 看跌孕线
- 前期蜡烛是看涨的（收盘价 > 开盘价）
- 当前蜡烛是看跌的（收盘价 < 开盘价）
- 当前蜡烛的实体完全包含在前期蜡烛的实体内
- 当前开盘价 < 前期收盘价
- 当前收盘价 > 前期开盘价

## 实现说明

**此形态使用TA-Lib的自适应阈值系统**来确定蜡烛特征，如"长实体"、"短实体"、"十字星"等。这些阈值会根据最近的市场行为自动调整，而不是使用固定百分比。

## 参数
- **输入**：OHLC数据（开盘价、最高价、最低价、收盘价）
- **输出**：形态识别数组（100表示看涨，-100表示看跌，0表示无形态）

## 含义
孕线形态表明：
- **看涨孕线**：下跌趋势后潜在的向上反转
- **看跌孕线**：上涨趋势后潜在的向下反转
- **犹豫不决**：显示市场犹豫和潜在趋势变化
- **确认**：应通过其他指标确认

## 使用方法
1. **反转信号**：在趋势极值处寻找孕线形态
2. **成交量确认**：通过成交量分析确认
3. **支撑/阻力**：在关键支撑/阻力位使用
4. **趋势背景**：考虑整体趋势方向

## 使用示例
```c
#include "ta_libc.h"

int main() {
    // 计算CDLHARAMI
    TA_RetCode ret = TA_CDLHARAMI(
        0,                    // startIdx
        99,                   // endIdx
        open_prices,          // open
        high_prices,         // high
        low_prices,          // low
        close_prices,        // close
        &outBegIdx,          // outBegIdx
        &outNBElement,      // outNBElement
        pattern              // outInteger
    );
    
    if (ret == TA_SUCCESS) {
        // 使用形态数组进行分析
        for (int i = 0; i < outNBElement; i++) {
            if (pattern[i] == 100) {
                // 看涨孕线形态
            } else if (pattern[i] == -100) {
                // 看跌孕线形态
            }
        }
    }
}
```

## 优势
- 显示市场犹豫不决
- 易于视觉识别
- 适用于所有时间框架

## 局限性
- 在震荡市场中可能给出虚假信号
- 需要其他指标确认
- 最好在关键支撑/阻力位使用
- 在快速移动的市场中可能滞后
