# CDL3LINESTRIKE - 三线打击形态

## 概述
三线打击形态（CDL3LINESTRIKE）是一种K线形态，由四根蜡烛组成，前三根在同一方向，第四根K线反转趋势。这是一个反转形态，可以预示趋势变化。

## 分类
形态识别

## 计算方法
CDL3LINESTRIKE 函数通过分析四根连续蜡烛来识别三线打击形态：

### 看涨三线打击判断标准
- 前三根K线：三只乌鸦（三根连续下跌的黑K线，每根开盘价在前一根实体内/附近）
- 第四根K线：白K线，开盘价低于第三根的收盘价，收盘价高于第一根K线的开盘价

### 看跌三线打击判断标准
- 前三根K线：三白兵（三根连续上涨的白K线，每根开盘价在前一根实体内/附近）
- 第四根K线：黑K线，开盘价高于第三根的收盘价，收盘价低于第一根K线的开盘价

## 实现说明

**此形态使用TA-Lib的自适应阈值系统**来确定蜡烛特征，如"长实体"、"短实体"、"十字星"等。这些阈值会根据最近的市场行为自动调整，而不是使用固定百分比。

## 参数
- **输入**：OHLC数据（开盘价、最高价、最低价、收盘价）
- **输出**：形态识别数组（100表示看涨，-100表示看跌，0表示无形态）

## 含义
三线打击形态表明：
- **看涨三线打击**：三根看跌K线后的强向上反转
- **看跌三线打击**：三根看涨K线后的强向下反转
- **趋势反转**：显示强反转动量
- **确认**：应通过其他指标确认

## 使用方法
1. **反转信号**：在趋势极值处寻找三线打击形态
2. **成交量确认**：通过成交量分析确认
3. **支撑/阻力**：在关键支撑/阻力位使用
4. **趋势背景**：考虑整体趋势方向

## 使用示例
```c
#include "ta_libc.h"

int main() {
    // 计算CDL3LINESTRIKE
    TA_RetCode ret = TA_CDL3LINESTRIKE(
        0,                    // startIdx
        99,                   // endIdx
        open_prices,          // open
        high_prices,         // high
        low_prices,          // low
        close_prices,        // close
        &outBegIdx,          // outBegIdx
        &outNBElement,      // outNBElement
        pattern              // outInteger
    );
    
    if (ret == TA_SUCCESS) {
        // 使用形态数组进行分析
        for (int i = 0; i < outNBElement; i++) {
            if (pattern[i] == 100) {
                // 看涨三线打击形态
            } else if (pattern[i] == -100) {
                // 看跌三线打击形态
            }
        }
    }
}
```
