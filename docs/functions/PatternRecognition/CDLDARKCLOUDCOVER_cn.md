# CDLDARKCLOUDCOVER - 乌云盖顶形态

## 概述
乌云盖顶形态（CDLDARKCLOUDCOVER）是一种K线形态，由两根蜡烛组成，其中第二根蜡烛开盘价高于第一根蜡烛的高点，收盘价低于第一根蜡烛实体的中点。这是一个看跌反转形态，可以预示趋势变化。

## 分类
形态识别

## 计算方法
CDLDARKCLOUDCOVER 函数通过比较当前和前期蜡烛来识别乌云盖顶形态：

### 乌云盖顶判断标准
- 前期蜡烛是看涨的（收盘价 > 开盘价）
- 当前蜡烛是看跌的（收盘价 < 开盘价）
- 当前蜡烛开盘价高于前期蜡烛的高点
- 当前蜡烛收盘价低于前期蜡烛实体的中点
- 当前蜡烛收盘价高于前期蜡烛的开盘价

### 公式
```
前期实体中点 = (前期开盘价 + 前期收盘价) / 2
当前收盘价 < 前期实体中点
当前开盘价 > 前期高点
当前收盘价 > 前期开盘价
```

## 参数
- **输入**：OHLC数据（开盘价、最高价、最低价、收盘价）
- **输出**：形态识别数组（100表示乌云盖顶，0表示无形态）

## 含义
乌云盖顶形态表明：
- **看跌反转**：上涨趋势后潜在的向下反转
- **卖出压力**：显示增加的卖出兴趣
- **阻力测试**：经常出现在关键阻力位
- **确认**：应通过其他指标确认

## 使用方法
1. **反转信号**：在趋势极值处寻找乌云盖顶形态
2. **成交量确认**：通过成交量分析确认
3. **支撑/阻力**：在关键支撑/阻力位使用
4. **趋势背景**：考虑整体趋势方向

        }

}
